# JMAM レポート出題傾向リファレンス

作成日: 2026-07-07。出典: ユーザーが受講した「ＭＫＳ昇格対象者コース (マネジメントエッセンスコース)」のJMAM Web School 採点結果レポート (メンバー指導 93点 / 労務管理 96点) と計数入門コースの問題レポート+公式解説。実際の添削講評・公式解説に基づくため、**作問の正誤根拠として教科書本文と同格に使ってよい** (教科書と矛盾する場合は教科書+法令を優先し、差分を記録)。

## 実試験の出題形式 (レポート課題ベース)

1. 正誤判定+誤り理由の記述 (各4点×5)
2. 穴埋め選択式 (各2点×10)
3. 用語説明・列挙 (各5〜10点)
4. ケース問題 (下線部の状況での管理職の行動を説明)

アプリの4肢正誤複数選択に落とす際は、「誤り選択肢」をレポートの誤答典型 (概念の入れ替え・限定しすぎ・逆順) から作ると実戦的。

## メンバー指導 — 確定した正誤ポイント

- 指導技術の基本ステップは 第一段階「物知りにしていく」→ 第二段階「行動変容を図らせる」。逆順は誤り
- カウンセリングの「受容」で得られるのは「カタルシス効果」(浄化)。「インサイト効果」は「自己一致」による内面的気づき。入れ替えは誤り
- ほめるのは特別な功績に限らない。メンバーが計画どおりに業務を遂行するという当たり前のことに対してほめる。いいと思ったらその場ですぐほめる (現場主義)
- 叱るのは特定の行為に限定。全人格を対象にするのは誤り (感情的な叱り方になり効果が失われる)
- 目標管理の面接はできるだけ小目標の単位で実行することが望ましい (正)
- OJTの成否は仕事の前の「Plan過程」、仕事の後の「See過程」でどれだけ有効なOJTを行っているかにかかっている
- 評価のプロセスとOJTのプロセスは逆順。OJTのプロセスは 能力 → 執務態度 → 業績
- マネジャーは自分の「権限」を周囲との「信頼関係」強化のために活用する
- OJT定義の穴埋め: 「職場組織内」で、熟練者が非熟練者に、知識・技術・経験を「直接」的に指導・伝授、「日常業務」を通じた訓練方法
- OJTの留意点: 知識・技能の伝授にとどまらず、能力開発の環境・条件を整備。模範解答の別視点: 日常業務の見直し・棚卸しで仕事を把握 → 育成必要点の明確化 → 育成プランにまとめる
- メンバーの言動を理解するための特徴4項目: ①時と状況と場所によって矛盾した言動をとる ②マネジャーの感情や動きに敏感に反応する ③他のメンバーと同じと思われる・扱われることに抵抗を示す ④気持ちを言葉より態度や行動で表現する。対処は「感情的にならず当たり前のこととしておおらかに構える」「メンバーをよく理解する」+**「メンバーを認める」**(1章-5。この要素の欠落で減点された)
- マネジメントの4側面: 仕事の管理・仕事の改善・メンバーの育成・職場の人間関係
- SOSサイン5つ: 言い訳をする・逃避する・攻撃する・無感動になる・退行する
- コンセプチュアル・スキルの説明には「広い視野からの課題の設計」の要素が必要 (4章-4)。「本質の抽出・問題発見・戦略的意思決定・目的の創造」だけでは1点減
- ケース対応の模範: Plan過程で目標・方針にメンバーを参画させ、期待度を明確にし、本人が自ら努力目標を設定することに合意する / やる気減退には「カウンセリング・マインドをもって面接」「話をよく聴く」「求めている支援を行い行動を変化させる」(2章-5〜7) / スキルアップ指導はテクニカル・スキル伝授に加え「あるべき姿を設計し現状とのギャップを明らかにして業務の見直しを指示」(4章-4・4-5)。「仕事を他者に任せる」だけでは不十分
- コンセプチュアル・スキルは一朝一夕には身につかないため早い段階から訓練する (C≧H＞T)

## 労務管理 — 確定した正誤ポイント

- 「手待ち時間」は使用者の指揮監督下でいつでも作業できる状態なら労働時間に含まれる。「自由に利用できる休憩時間」とする説明は誤り (手空き時間との区別)
- 終業後の勉強会でも出欠が賞与査定等に反映されるなら労働時間。朝礼・体操も強制参加や不参加が遅刻扱い・人事考課反映なら労働時間
- セクハラは2007年4月施行の改正男女雇用機会均等法で男性労働者も保護対象
- 事業場外みなし労働時間制: 携帯電話等で随時指示を受ける場合は適用外。ただし携帯所持が急用連絡用なら適用可
- 配置転換・転勤: 就業規則に定め+雇入れ時に就業場所・業務の変更の範囲を書面明示 (正)。職種明示がない場合、業務上必要と認められれば同意なく配転を命じられるが、業務上の必要性がない・不当な動機目的・著しい不利益の場合は権利の濫用で無効
- 休憩: 1日の労働時間が8時間を超える場合1時間以上、6時間以下なら不要 (45分は6時間超8時間以下)
- 年休: 労働者側=時季指定権、使用者側=時季変更権。時季変更権は「事業の正常な運営を妨げる」客観的理由がある場合に限る。スケジュール変更の手間は理由にならない。就業規則の届出ルール (前日まで) が基本的に優先
- 派遣の三者関係: 派遣元-労働者=雇用関係、派遣先-労働者=指揮命令関係、派遣元-派遣先=労働者派遣契約
- 割増賃金 (月45時間・年360時間以内の前提): 時間外25%以上、時間外が深夜 (22時〜5時) におよぶと50%以上、法定休日労働35%以上
- 変形労働時間制の説明に必要な2要素 (欠落で減点された): 「残業手当の増加を抑えることができる」「残業命令を出さなくても1週40時間1日8時間を超えて労働させることができる」。1ヵ月単位で定める事項: 週平均40時間以内・各日各週の労働時間・各日の始業終業時刻・起算日
- 対価型セクハラ=性的言動への対応により解雇・減給・降格等の不利益 / 環境型=就業環境が不快になり能力発揮に重大な支障
- 労使協定=過半数組合 (なければ過半数代表者) と締結した書面による協定。休憩・賃金支払・変形労働時間制・育児介護休業などで必要
- コアタイム=必ず労働しなければならない時間帯 (特段定めなくてもよい)。フレキシブルタイム=労働者の選択によって労働できる時間帯
- ノーワーク・ノーペイの原則: 働かなかった分は賃金請求権なし・支払義務なし。フレックスの清算期間の考え方の基礎

## 計数入門 — 確定した正誤ポイント・計算公式 (公式解説より)

正誤:
- 計数感覚: 計算が速い・簿記会計の知識だけでは不十分。分析力があり数字を活用して成果をあげられること
- 「売上はこれだけ→利益はこれだけ必要→費用はこれだけ」は目標利益的発想 (財務会計発想ではない)
- 生産性=投入量に対する産出量 (正)
- 左に資産・右に負債と純資産=貸借対照表。損益計算書はプロセスごとの利益で収益力を表す
- 総合原価計算=見込生産方式。一品一品異なる注文生産は個別原価計算
- 特定製品だけに発生と判別できる費用=製造直接費 / 判別できない=製造間接費 (入れ替えが定番の誤り)
- 標準原価=科学的・統計的調査にもとづく合理的な原価 (正)。標準原価は現場担当者にも背景・手順とともに知らせて納得してもらう (隠すのは誤り)
- 操業度差異=生産量の調整により発生する原価差異 (作業能率の悪さではない。それは能率差異)。賃率差異=予定より高い人件費の人を使用したことによる差異
- 費用分類: 材料費に燃料費 / 労務費に給料 / 支払経費に外注加工費 / 月割経費に減価償却費 / 測定経費に電力料・ガス・水道 / 発生経費に棚卸減耗費・仕損費
- 財務三表=貸借対照表・損益計算書・キャッシュフロー計算書 (法人税申告書は含まない)
- 原価計算の目的: 正しい決算書作成・販売価格決定・原価管理・採算計算 (「資金の流れの確認」は含まない)
- 固定費=売上や操業度に関係なく一定に発生する費用
- 損益分岐点比率100%以上=赤字経営
- 経営判断で増減する分の原価=増分原価。すでに支出済みで判断に影響しない原価=埋没原価
- ポートフォリオ分析=クロス分析、2つの視点 (正)
- ベンチマーキング: 自社内に限らず同業他社・異業種・世界のエクセレントカンパニーまで対象
- ボトルネック=ラインバランス効率を阻害する要因 (効率そのものではない)
- 季節指数の精度には最低3年間の実績が必要 (2年間では不足)
- 固定費の変動費化: 経営の耐久力を高める。パート化・レンタカー化・外注化が例
- 原価以下の特別受注の3前提: 製造能力に余力・既存マーケットに影響を与えない・利益率より利益額が優先課題。「自社にしか製造できない」は含まれない
- 赤字製品廃止の判断ポイント=限界利益 (貢献利益)
- 重点管理にもっとも適した分析=ABC分析
- Zチャート: 右上がり=売上増加傾向・並行=横ばい・右下がり=減少傾向。幅が広い=好調・狭い=低調

計算公式 (数値例は公式解説の検証済み値):
- FCF = 営業CF + 投資CF (550 + (-300) = 250万円)
- 先入先出法: 先に購入した材料から払出す (100円×90個=9,000円)
- 配賦: 間接費×配賦基準比率 (作業時間2:1 → 3,000千円×2/3=2,000千円)
- 経常利益 = 営業利益 + 営業外収益 − 営業外費用 (140+60−50=150万円)
- 完成品原価 = インプット総額×完成数量/(完成+仕掛) (1,020,000×230/250=938,400円)、仕掛品原価=×20/250=81,600円
- 価格差異 = (実際単価−標準単価)×実際使用量 ((250−200)×7,500=375,000円)
- 数量差異 = (実際使用量−標準使用量)×標準単価 ((7,500−5,000)×200=500,000円)
- 損益分岐点売上高 = 固定費 ÷ (1−変動費率) (3,000÷(1−0.6)=7,500万円)
- 安全余裕率 = 100 − 損益分岐点比率 (100−75=25%)
- 必要売上高 = (固定費+目標利益) ÷ (1−変動費率) ((3,000+1,800)÷0.4=12,000万円)
- 定額法減価償却費 = 取得価額×(1/耐用年数) (200万×0.200=40万円/年)
- 発注日の適正在庫 = リードタイム日数×1日平均販売数量 + 安全在庫 (2×5+3=13個)
- ラインバランス効率 = 作業時間合計 ÷ ボトルネック拘束時間 ×100 (30/40=75%)
- 小売売上高 = 客数×買上率×1人あたり買上点数×1点あたり平均単価 (500×0.3×20×200=600,000円)
- 季節指数 = 当月の3年合計 ÷ 3年間総合計 ×100 (1,240/11,740×100≒10.6%)
