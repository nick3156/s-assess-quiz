# 序章 計数感覚とはなにか

## この章で学ぶこと

本コースは、計数感覚が身についた「数字に強い」ビジネスパーソンになることをめざして学習を進めていきます。

この章では、「計数感覚とはなにか」という疑問を解決し、数字に強いビジネスパーソンになるためには何を学んだらよいかを確認します。この章の学習で、本コースの全体像を理解しましょう。

| 序-2 | 数字に強いビジネスパーソンになるには |

## 序-1 ビジネスにおける「数字」の重要性

### 学習のポイント

- 具体的な数値を使うと、相手にも的確に状況を説明することができる。
- 企業経営では、数字をタイミングよく把握して活用することがとても重要である。
- 計数感覚に優れた人とは、数字を使って問題を発見し、成果をあげることのできる人である。

### 1 ビジネスにおける「数字」の重要性

「計数感覚」とは、どんなものでしょうか。身近な例で考えてみましょう。

Aさんが、好きなプロ野球選手について、友人にどこがすばらしいのかを説明しようとしたとします。「彼は打撃のセンスがすごいんだ!」とか、「とても活躍している選手なんだよ!」などと説明して、その選手のよさが伝わるでしょうか。ある程度その選手が出場する試合を見ている人でなければ、それでは納得できないでしょう。しかし、「今シーズンの打率は3割3分で首位打者なんだよ」と説明したらどうでしょう。野球のルールを知っている相手であれば、どれくらいすごい選手なのかすぐにわかってもらえるはずです。

ビジネスも、これと同様です。上司から「今月の販売状況はどう?」と聞かれたとき、「売上は伸びています」と答えたらどうでしょうか。あるいは「仕事の進み具合は順調なの?」と聞かれて、「ほぼ順調ですが、少し遅れています」と答えたらどうでしょうか。

「売上は伸びています」や「ほぼ順調です」、「少し遅れています」という返答では、どの程度伸びているのか、どのように順調なのか上司にはまったくわかりません。これでは状況が把握できず、アドバイスもしようがないでしょう。慎重な上司であれば、「本当に順調なのだろうか」と不安になってしまうかもしれません。

では、どう回答すれば的確に状況を伝えられるのでしょうか。ここで、「数字」の出番なのです。具体的な数字を含んだ表現にしてみましょう。「売上高は前年比で3%アップです」、「予定より少し遅れていますが、あと3日で完成します」といえば、だれでも客観的に状況を把握できます。

このように、数字は日常の業務を滞りなく進めるうえでも大きな役割を果たしていますが、「会社の経営」となると、その重要度はさらに増します。企業は利益を獲得しなければ存続することはできません。目標とする売上や利益を獲得するため、従業員全員が知恵をしぼり必死に努力しています。目標となる売上や利益を数字でしっかり示すことで、従業員全員の努力を1つの方向にまとめることができるのです。

また、経営に関わる数字はタイミングも重要です。前月の売上や利益の計算が翌月の10日を過ぎてから明らかになるようでは問題です。たとえば、なにか大きな問題が起こり、目標達成が困難な状況となっていたら、対策を行うための時間を10日以上もムダにしてしまったことになります。

的確なタイミングで数字を把握することは、健全な経営のために不可欠なのです。このように企業の経営において数字はもっとも基本的なものであり、これを上手にタイミングよく活用することが、大変重要な課題といえるでしょう。

### 2 計数感覚に優れた人ってどんな人?

「○○さんは数字に強い人だ!」とか、「計数感覚に優れている」といった表現を聞くことがあるでしょう。「数字に強い人」というと、「仕事ができる」というイメージがあり、優秀なビジネスパーソンの代名詞に使われることが多いようです。

では、「数字に強い人」とはどんな人なのでしょうか。計算が速く正確にできること、あるいは簿記や会計の知識をもっていることなどが、計数感覚だと思っている人もいるかもしれません。しかし、それはちょっと違います。たしかに、計算間違いが多い人、計算の遅い人は「数字に強い人」とはいえないかもしれません。計算が速いことや、簿記・会計の知識があることは、数字を理解するのに有利ではあるでしょう。しかし、それだけでは十分ではないのです。

#### 1 数字を分析できる人

先ほどのプロ野球選手の説明について、もう少し考えてみましょう。「打率は3割3分である」というだけでなく、「全体の打率は3割3分だが、左投手だと3割8分も打つ」とすれば、左投手に対してもっと好成績であることがわかります。

また、販売状況について「売上高は前年比で3%アップです」と回答した例でも、「そのうち新規顧客分で全体の5%を伸ばしましたが、既存顧客では2%マイナスでした」とその内容をいくつかの特性に分けて報告したらどうでしょう。3%伸びた要因や、既存顧客の実績は不振であるということもわかります。

このように全体をひとくくりで見るのではなく、特別な意味をもったいくつかの層（これを特性といいます）に分けて見ることを「分析」といいます。

数字の全体をまとめて説明するだけでなく、特性に分けて加工し、背後にある特徴や問題点を浮かび上がらせることができる人は「分析力のある人」といわれます。「分析力のある人」は「計数感覚に優れた人」だと言い換えることができます。

#### 2 分析した数字を活用して成果をあげられる人

今度は、身近にいる「計数感覚に優れた上司や先輩」について考えてみましょう。

数字に強い人は、どの業種、どの職種にもいます。利益率の高い店舗と自店舗の損益や生産性を比較して、自店舗の問題点を調べ、利益率を向上した店長がいます。「この製品の原価は高すぎる!」と問題提起し、原価計算の矛盾を指摘した製造部の社員がいます。原価以下の特別注文が入り、全員が受注に尻込みするなかで「この特別注文に応じたら利益は増える!」と読んで、実際に利益を増やした先輩もいます。また、万年赤字の営業所を担当し、「あと○○パーセント売上を伸ばせば黒字になる!」と目標を示して社員の意欲をかきたてて、本当に黒字にした営業所長もいます。

計数感覚に優れた人は、数字を駆使して問題を発見し、実際に成果をあげることのできる人なのです。

具体的にいうと、利益率を高めた店長や、原価計算の矛盾を指摘した社員のように、数字をみて異常や問題に気づく人、特別注文に応じて利益を増やした社員や、赤字の営業所を黒字にした営業所長のように、先の数字を読める人、さらに数字を活用して目標を立て、成果をあげられる人、そんな人が計数感覚に優れた人ということができるでしょう。

**計数感覚に優れた人とは**

| レベル | 内容 |
|---|---|
| Level 1 | 数字を使って状況を説明できる人 |
| Level 2 | 数字を特性ごとに分けて問題点を見つけられる人 |
| Level 3 | 数字をみて異常や問題に気づく人 |
|  | 先の数字を読める人 |
|  | 数字を活用して目標を立て、成果をあげられる人 |

## 序-2 数字に強いビジネスパーソンになるには

### 学習のポイント

- 計数感覚は、基本から学習することでだれでも身につけることができる。
- 計数感覚を身につけるためには、「利益意識、コスト意識を高める」「数値計算のしくみを理解する」「コストの分析力を高める」「問題の発見力・改善力を高める」という順で学習するとよい。

### 1 数字に強くなるためのプロセス

どうすれば計数感覚に優れた「仕事のできる」ビジネスパーソンになれるのでしょうか。

「数字に強い」といわれる人のなかには、きちんと学習したと思えないにも関わらず、天才的な計数感覚をもった人もいます。自ら創業して成功した経営者に多いようで、いちから商売を積み重ねるなかで、自分流の計数感覚を磨き上げているのでしょう。

しかし、もともとのセンスや創業の経験がなくても、基本からしっかり学習すれば、計数感覚を身につけることはできます。図表序-1は、優れた計数感覚をもった、できるビジネスパーソンになるまでのプロセスを示したものです。そのプロセスについて、ここで確認しておきましょう。

**図表序-1 | 計数感覚を身につけるためには**

| 段階 | 学習テーマ | 確認すること |
|---|---|---|
| 3 | 問題発見力・改善力を高める | (1) 改善・改革の計画を立案し、実行する |
| 3 | 問題発見力・改善力を高める | (2) 数値の異常、問題に気づく |
| 2 | コストの分析力を高める | (1) 活動の変化に対してコストがどう動くか |
| 2 | コストの分析力を高める | (2) 現場でどんな活動が実施され、どんなコストが発生しているか |
| 1 | 数値計算のしくみを理解する | (1) 損益分岐点、各種の採算計算のしくみ |
| 1 | 数値計算のしくみを理解する | (2) 利益計算や原価計算のしくみ |
| 前提条件 | 利益意識、コスト意識を高める | 利益やコストに関心をもつ |

#### 1 前提条件としての利益、コスト意識

計数感覚を身につけるためには、利益やコストに関する高い意識をもっている必要があります。

企業活動の最終的な目標は利益をあげることにあります。利益はどのようにつくられているのか、そのしくみを知り、より高い利益をあげることを意識することが大切なのです。たとえば、設備を新しく購入する場合には、「この設備を導入すれば仕事がしやすくなる」と説明するだけでは、利益意識、コスト意識がないといわれてしまうでしょう。利益意識、コスト意識が備わっていれば、設備を新しくすることによっていくら利益が増えるのか、あるいは生産性（たとえば時間あたりの仕事の効率）がどれだけよくなるのか、といった点を説明できるはずです。

また、自分の担当の仕事で大幅な赤字を計上している場合に、「私の仕事は社会にとって大変価値のある仕事だ、大切なのは利益ではない」という人がいるとしたら、これも利益意識、コスト意識が欠如しているといえるでしょう。このような場合、他の人が稼いだ利益をもとに自分の仕事が成り立っていることを自覚し、それを感謝しなければならないのです。

#### 2 数値計算のしくみを理解する

会社で使用する数字には、それぞれ算出のしくみがあります。たとえば利益計算は、複式簿記のしくみにもとづいて行われますし、製品の原価は大変細かい原価計算のしくみ（第2章で学習します）によって計算されています。

このような計算は、すべてルールが決まっており、機械的に算出されると思う人が多いかもしれませんが、実はそうではありません。

利益の計算を例にとると、売上代金の貸倒損失や、不良在庫があった場合には、決算時に損失額の見積りが必要になり、そこには人の判断が介入します。

このような利益計算のプロセスを知っている人は、計上された損失額をみて、「この判断は甘い、損失額は実態に比べて少なすぎる」という指摘をすることもできるでしょう。

「この製品の原価は高すぎる!」と指摘した製造部の社員の例では、原価計算のプロセスで人の判断が介入することも知っているからこそ、その部分の問題を指摘できたのです。

また、「あといくら売ったら利益が出るか」という問題は、損益分岐点（第3章で学習します）という計算のしくみからわかりますし、「原価以下の特別注文に応じた場合に利益が出るか否か」という問題は、採算計算のしくみ（第3章で学習します）から判断することができます。

計数感覚に優れた上司や先輩が異常や問題に気づいたり、先の数字を読めたりするのは、このような基本的な計算のしくみを理解しているからなのです。

#### 3 コストの分析力を高める

「コストの分析力」は、経営用語としては「アクティビティ分析」と呼ばれます。アクティビティ分析は、コストと活動の因果関係を明確にしてコスト管理を行うための分析手法です。

これは計数感覚を身につけるうえで大変重要な考え方ですので、ぜひつぎの2つの視点を理解しておきましょう。

**コストが違えば背後にある活動も異なる**

コストや売上などの数字は人の活動の結果を表したものです。具体的にいえば、同じ仕事でもやり方（活動の方法）が悪ければコストは高くなり、効率よく進めれば低くなります。数字の背後には必ず活動があるのです。

たとえば、人員構成と売上高が同じ2つの営業所でコストが異なる場合、営業方法や管理業務など、必ずなにかしらの仕事の進め方（活動）が異なっているはずです。コストを削減するためには、その違いを突きとめ、活動のしかたを変えなければなりません。

**活動が変化したらどんなコストが動くか**

先ほどの例の裏返しではありますが、活動が変化すれば、コストも変化します。たとえば、製造部門で機械の稼働時間を1時間増加させたとします。このとき、コストはどのように変化するでしょうか。電気代や燃料などの動力費は余計に発生します。材料の使用料も増えるでしょう。

人件費はどうでしょうか。作業が増えた分、人件費も増えるような気がします。しかし、社員だけで作業をした場合には給料は定額であり、人件費は増えないかもしれません。ただし、その1時間の作業を残業で行ったり、時給制のパート社員が作業したりすれば人件費は増加します。

「コストの背後にある活動の実態を把握する」、「活動が変化したらどんなコストが動くか理解する」という2つの視点は、コスト発生のもとである業務の実態を把握するという意味では同じことであり、双方とも計数感覚を身につけるうえでもっとも重要となるものです。

#### 4 問題の発見力・改善力を高める

これは数字をみて、それがよいのか悪いのか、正常なのか異常なのかを判断する能力のことです。

この能力を養うには、さまざまな数値計算のしくみを理解し、さまざまな業務の実態を知って、活動やコストの分析力を鍛える必要があります。また、優良企業の利益率や原価率、あるいは自社の業績のよい営業所の利益率や経費率などの指標について常に関心をもつことも大切です。そうすることで、次第に数字に対する判断力は向上してくるものです。

さらに、いくら数字の分析をして問題をみつけることができたとしても、活動内容を改善・改革しなければ、業績は向上しません。仕事のできる上司、先輩は、単に計数感覚に優れているだけでなく、仕事の改善・改革を実行し、業績を向上させる行動力がある人でしょう。

計数分析の最終的な目標は業績向上です。これを達成するためには業務（活動）改善に積極的にチャレンジしていかなければなりません。

計数を活用してどのように業務の改善に結びつけるかについては第4章で学習しますが、いきなり業務改善をしようと思っても、そんなにすぐに成功するものではありません。いまのうちから、自社内のさまざまな改善事例や同業他社の事例、あるいはエクセレント企業の改善事例に関心をもち、なにを改善してどのような成果が出たのか、そのときにどんな工夫や苦労をしたのか、確認しておくとよいでしょう。

そのような事例を日々集めることによって、計数感覚が身につくだけではなく、改善マインドが旺盛になり、仕事のできるビジネスパーソンへと成長できるはずです。

> 本コースの学習で、計数感覚を身につけよう!

## 1-1 利益意識を高めよう

### 学習のポイント

- 計数感覚を身につけるには、売上高だけでなく利益率に関心をもつことが重要である。
- 日本企業の利益率について、その水準を理解する。
- 利益は「収益 - 費用 = 利益」という計算で求められ、これを損益計算と呼ぶ。

### 1 100円売るといくらもうかる?

計数感覚を身につけるためには、日ごろから利益について関心をもつことが重要です。あなたの利益についての意識は十分でしょうか。まずは、つぎの問いについて、考えてみてください。

**Q. 平均的な日本の企業では、100円の売上で、いくらの利益を得ることができるでしょうか?**

選択肢: 1. 30円 / 2. 20円 / 3. 10円 / 4. 5円 / 5. 3円 / 6. 1円

#### 1 業種別の利益を確認しよう

図表1-1は、財務省から公表されている法人企業統計から、2020年度と2024年度の日本企業の業種別、規模別の売上高営業利益率を抜粋したものです。

2024年度の売上高営業利益率を業種別に見ると、製造業では電気機械製造業、非製造業ではサービス業がそれぞれ7.1%、8.1%と高い水準の利益率を上げています。一方で、食料品製造業や卸、小売業は3.6%、2.4%と低い水準になっており、業種によってバラツキがあることがわかります。全産業の平均で見ると、2024年度の売上高営業利益率は5%と、全体的に企業業績は好調です。なお、2020年度は3.1%でした。

この数値を単純に見ると、「100円売って3-5円の利益」と思われがちですが、実際にはそこから法人税が差し引かれます。大企業では利益の約30%、中小企業では約20%が法人税等として課税されるのが一般的です。法人企業統計では、税引き後利益に関する数値は公表されていませんが、売上高営業利益率から売上高純利益率を推定すると、日本の企業は「100円の売上に対し、最終的な利益は2-4円」の水準であると考えられます。

また、法人企業統計では、企業規模を資本金で区分しています。資本金10億円以上の大企業では、2024年度の営業利益は7.8%と高水準を確保していますが、規模が小さくなるほど利益率は低下しています。特に、資本金1,000万円未満の企業では、2024年度は1.5%の利益率となっていますが、2020年度は利益が出ない、赤字の状態です。実は、この規模の企業の利益率が黒字になったのはごく最近であり、2022年度以前はずっと赤字の状態であったのです。

このような統計を見ると、企業が利益を獲得することがいかに難しいか実感してもらえると思います。まさに100円の売上に対して1円が、企業の命運を握っているといえるでしょう。

### 2 利益を意識しよう

ここまで、利益率について業種別の平均やトップ企業を例にとって確認してきました。いかに利益をあげることが難しいか、実感できたのではないでしょうか。

ふだんの業務では、職種によっては売上高にばかり意識がいきがちな人もいるかもしれません。しかし、企業は利益を獲得しなければ存続することはできません。目標とする利益を獲得するため、全員が知恵をしぼり必死に努力しているのです。

企業活動の最終的な目的は、「利益をあげること」にあります。利益を多く出してこそ仕事であり、企業活動なのです。利益に対する意識を常に忘れないようにしましょう。

では、その利益はどのように計算されるのでしょうか。ここで「利益」について、その算出方法を確認しておきましょう。

会社の利益は、一定期間（通常1年）の間に獲得した収益から費用を差し引いて計算されます。

```text
収益 - 費用 = 利益
```

#### 1 収益とは?

収益とは、経営活動によって獲得した成果のことです。その主なものは、会社の本業である商品・製品の販売やサービスの提供によって獲得した売上高です。

売上高以外にも、預金や社債の利息、所有する株式から得られた配当金による収入、土地や建物を貸して得られる不動産収入、株式や土地等の売却益など、本業以外の稼ぎも収益となります。

#### 2 費用とは?

費用とは、収益を獲得するために消費された経済価値のことです。

その主なものは、売り上げた商品・製品、サービスの原価や、その販売段階で要した販売促進費、配送費、人件費、営業所や本社の管理費などです。これ以外にも、銀行からの借入金に対する支払利息、株式や土地などの売却損なども含まれます。

#### 3 収益と収入、費用と支出の違い

「収益」には「収入」、「費用」には「支出」という似たことばがあります。似てはいるのですが、それぞれ意味は異なります。

たとえば、ある決算期間内に広告宣伝を行ってその代金を支払ったとすると、その代金はその決算期間内の「費用」であり「支出」となります。しかし、広告宣伝を行ったけれど、まだ代金の支払いは行っていないということもあるでしょう。このような場合、その期間内の「費用」は発生していますが、「支出」はありません。

このように、支払いが行われなくても、広告宣伝というような経済的な価値が消費された事実があれば、その期間の費用になります。

また、収益についても同様です。商品を販売していても、その決算期間内で代金を回収していないことがあります。この場合には、その決算期間では売上という収益が計上されますが、収入にはなりません。

#### 4 損益計算とは?

収益 - 費用 = 利益ですから、収益が費用を上回れば利益が出て、収益よりも費用のほうが上回れば損失が出ます。この計算は、損失が出ることがあれば利益が出ることもあるため、損益計算と呼ばれます。

**図表1-3 | 損益計算における利益と損失**

| 状態 | 図の読み取り |
|---|---|
| 収益 > 費用 | 収益の枠が費用を上回り、差額として利益が出る。 |
| 費用 > 収益 | 費用の枠が収益を上回り、差額として損失が出る。 |

## 1-2 コスト意識を高めよう

### 学習のポイント

- 経済の成長率が低い成熟期の時代には、目標利益的発想で考える必要がある。
- コスト意識をもつための第一歩として、自分の人件費を意識するとよい。
- 在庫を過剰にもつと、さまざまなコストが発生し、経営を圧迫することもある。

### 1 コスト意識とは?

前節で、利益は収益から費用を差し引いたものであることを学習しました。つまり、利益を大きくするためには、収益を多く獲得するとともに、なるべく費用（コスト）を少なくするとよいということがいえます。

仕事に取り組むときは、常にその活動にかかるコストについて考える癖をつけましょう。それが「コスト意識」なのです。

現在のように経済の成長率が低い成熟期の時代には、一般に「著しい売上の増加は期待できない」という状態が続きます。このような時代では、「少しでもコストを削って利益を多く捻出する」という考えではなく、「目標利益的発想」で考えることが大切です。

| 考え方 | 計算式 |
|---|---|
| 財務会計の計算方法 | 収益 - 費用 = 利益 |
| 目標利益的発想による利益意識 | 収益 - 利益 = 費用 |

目標利益的発想とは、「売上はこれだけしか望めない、また会社としてこれだけの利益が必要だ、だから費用はこれだけしか使えない!」という考え方です。限られたコストの範囲内で、知恵を出し計画を立てて日々の活動を行わなければなりません。

### 2 自分にかかるコストを意識しよう

コストを常に意識するといっても、はじめは難しいかもしれません。第一歩として、「自分の人件費を自分で稼ごう」と考えると、イメージがわくのではないでしょうか。

ひと口に「人件費」といっても、そのなかにどのような費用が含まれているか、正確に理解している人は少ないものです。

人件費は、給与と賞与の合計額ではありません。もしそう思っていた人がいるとしたら、ここでしっかりと自分にかかっているコストを確認しましょう。

**人件費に含まれる費用**

| 区分 | 内容 |
|---|---|
| 給与・賞与 | 残業代も含んだ給与・賞与 |
| 退職金 | 勤務期間全体で積み立てられている退職金 |
| 法定福利費 | 健康保険、厚生年金、介護保険、雇用保険、労災保険の会社負担分など、法律で定められている福利厚生にかかる費用。広義の社会保険料 |
| 法定外福利費 | 家族手当、住宅手当、交通費、慶弔手当など、会社が任意で実施する福利厚生関係の手当 |

#### 1時間あたりの人件費を計算しよう

若手社員Aさんの人件費は、月給の約何倍でしょうか。

**Aさんの勤務条件**

| 項目 | 金額・条件 |
|---|---|
| 月給 | 250,000円 |
| 賞与 | 月給4カ月分 |
| 1カ月あたりの残業代 | 20,000円（10時間） |
| 1カ月あたりの通勤費 | 10,000円 |

選択肢: 1.3倍 / 1.5倍 / 1.7倍 / 1.9倍

問いの答えを出すために、まず、1カ月あたりの人件費がいくらになるか、それぞれの費用を確認します。月給、残業手当、1カ月あたりの通勤費は勤務条件に明記されていますから、それ以外について計算していきます。計算にあたっては、全産業の平均値とします。

賞与は、月給の4カ月分で1年分ですから、1カ月分は `250,000円 x 4カ月 / 12カ月 = 約83,000円` となります。

退職金は、全産業の平均値では、33年間の勤続年数で退職金は約20,000,000円です。1カ月あたりに換算すると、`20,000,000円 / 33年 / 12カ月 = 約50,000円` となります。

法定福利費は、月給250,000円であれば、1カ月あたりの会社負担分は、おおよそ健康保険料が12,000円、厚生年金が20,000円、雇用保険などが2,000円くらいです。

法定外福利費は、1カ月あたり、住宅手当として10,000円、その他に15,000円が支給されているとします。

| 人件費項目 | 内訳 | 金額（円） |
|---|---|---:|
| 給与 |  | 250,000 |
| 賞与 |  | 83,000 |
| 残業手当 |  | 20,000 |
| 退職金 |  | 50,000 |

また、月の残業時間込みの勤務時間を170時間とすると、1時間の人件費は約2,800円になります。

この数値は、全産業の平均値をもとに作成しています。福利厚生の充実度は会社によって異なり、給与・賞与以外の項目にもっと高いコストがかかる会社もあるかもしれません。ぜひ図表1-4を活用して、自分の1カ月あたり、また1時間あたりの人件費を計算してみてください。また、その際にはあなたにかかっているお金は、人件費だけでなく、パソコンや机等の事務費、さらには採用に関する費用もあることも頭に入れておきましょう。

### 3 在庫にかかるコストを理解しよう

在庫についての考え方も、コスト意識が重要なものの1つです。「在庫を減らそう」という目標を聞いて多少の会計知識をもつ人は、「在庫は資産だから、どれだけ抱えたとしてもコストは発生しないのではないか」と疑問に思うかもしれません。また、お客さまの要望に確実に対応するためには、在庫に余裕をもたせたほうが有利だという面もあります。ではなぜ、在庫を多くもつことが問題なのでしょうか。

ここで在庫のコストについて、基本的な考え方を学習しておきましょう。在庫は棚卸資産とも呼ばれ、つぎのような種類があります。

**在庫の種類**

| 種類 | 内容 |
|---|---|
| 商品 | 他社から仕入れてそのまま販売するもの |
| 製品 | 自社で製造加工して完成したもの |
| 仕掛品 | 製造途中にある未完成品 |
| 原材料 | 製造して製品に組み込まれるもの |
| 貯蔵品 | 消耗品や消耗工具備品など |

#### 1 在庫が多くなる要因

在庫が必要な理由は、もちろん品切れをしないためです。もしも品切れしてしまうと、売上の機会を逃してしまう（これを機会ロス、またはチャンスロスといいます）ばかりでなく、相手先に迷惑をかけることになり、信用を失ってしまうかもしれません。

また、小売業ではたくさんの種類の商品を陳列することで、お客さまが商品を選びやすくなるという効果もあります。この場合、在庫を増やすと売上が増えるという考え方から、在庫を多くもつ傾向があります。

このような理由から、在庫が適正な量よりも多くなってしまっているということがよくあります。

しかし、在庫を多くもつことは、経営につぎのような影響を与えます。

**在庫が経営に与える影響**

| No. | 影響 |
|---|---|
| 1 | 金利費用 |
| 2 | 保管費用、管理費用 |
| 3 | 在庫ロス（減耗分） |
| 4 | 値引き、廃棄ロス |
| 5 | 経営資金の圧迫 |

#### 2 在庫のコスト

**1. 金利費用**  
在庫を購入する資金を借入金で調達した場合、金利がかかります。

**2. 保管費用、管理費用**  
在庫には、保管するための倉庫料、倉庫担当者の人件費、フォークリフトなどの設備の経費がかかります。また、在庫を管理するためのデータ処理費や帳票代などの管理費用も発生します。

**3. 在庫ロス（減耗分）**  
在庫には、どのように注意しても取扱ミスや自然劣化による破損、汚損がある程度発生します。また、紛失や盗難の可能性もあります。在庫が増えると在庫の管理がずさんになりやすく、これらの在庫ロスの発生率は増加する傾向があります。

**4. 値引き、廃棄ロス**  
過剰な在庫をもつことのもっとも大きなリスクは値引き、廃棄ロスです。これは、売れなくなった製品の在庫が残ってしまい、値引きや廃棄処分を行うことによって起こるロスです。製品にはライフサイクルがあり、人気商品でもやがて売上が落ちます。多くの在庫をもてば、このロスも大きくなるのです。

**5. 経営資金の圧迫**  
1から4までの項目は、損益計算に影響する費用ですが、それ以外に経営資金にも影響があります。在庫をもっていることで、その分の資金が活用できなくなってしまうのです。たとえば、1,000万円の在庫をもつと、1,000万円の現金がそこに寝ていることになります。

在庫に余裕をもたせると、欠品の減少や、まとめ買いによるコストの削減が望めるでしょう。しかし、過剰在庫が原因で、費用が増えたり資金面の負担が大きくなったりして、経営破綻に追い込まれた企業もたくさんあります。過剰に在庫をもたず、品切れなどの機会ロスを発生させないよう、適正在庫を維持することが重要です。

※適正在庫: 品切れを起こさず、販売数に応じた数量の在庫数量のこと。4-2-2で学習します。

### 4 ムリ、ムダ、ムラを発見する

**ムリ、ムダ、ムラとは**

| 用語 | 意味 |
|---|---|
| ムリ | 従業員や機械設備等が有する能力以上の負荷がかかっている状態 |
| ムダ | 従業員や機械設備等が有する能力以下しか使いこなせずにいる状態 |
| ムラ | 処理すべき作業量や処理能力に偏りがあり、活動成果にバラツキが生じること |

それぞれについて、詳しくみていきましょう。

#### 1 ムリとは?

日常の仕事のなかには、つぎのようなムリがあります。

- 疲れる仕事のムリ: 不自然な姿勢での作業、暑すぎたり寒すぎたりするなかでの作業などのこと。
- 長時間労働のムリ: 休憩を取らずに作業を続けたり、残業をやりすぎて集中力がなくなったりすること。

このほかにも、機械酷使のムリ、生産計画のムリなど、さまざまなムリが発生します。このようなムリがあると、一時的には効果があるものの、継続すれば余分な修理費や不良品が発生してしまい、結局コストが増加します。

#### 2 ムダとは?

日常の仕事のなかには、つぎのようなムダがあります。

- 歩くムダ・探すムダ: 作業に必要な道具を取りにいくためにあちこち歩き回ったり、探したりする時間などのこと。
- 迷うムダ: 工具や車両をどこに置いたらよいか迷ったり、手に取った材料の品名が何か迷ったりする時間などのこと。
- 記入するムダ: 記入する書類が何枚もあったり、口頭での連絡ですむことを書類へ記入したりする手間などのこと。
- 手待ちのムダ: 仕事ができる状態なのに、前の工程でとまどっていて、手待ち状態になっていること。

このほかにも、協力しないムダ、作りすぎのムダ、不良のムダ、在庫のムダ、運搬のムダ、加工のムダ、動作のムダなど、さまざまなムダがあります。

#### 3 ムラとは?

ムラは、ムダな状態とムリな状態が両方含まれていることです。日常の仕事のなかには、つぎのようなムラがあります。

- 精神的なムラ: あるときはやる気があり、あるときはまったくない、というような状態のこと。
- 仕事配分のムラ: あるときは忙しく、あるときはひまである、というように仕事量にばらつきがあること。
- 生産数量のムラ: 曜日や時間帯によって、生産数量が異なる状態のこと。

このほかにも、在庫量のムラ、人員配分のムラなどがあります。仕事を効率よくこなすには、この「ムリ」「ムダ」「ムラ」を発見し、原因を徹底的に究明し、排除しなければなりません。

ここで説明した「ムリ」「ムダ」「ムラ」、あるいは「3ム」「3M」は、特に製造業にとっては基本的な用語です。意味を理解するだけでなく、職場の「ムリ」「ムダ」「ムラ」に気づく感性を養ってください。

> ムリ、ムラ、ムダがないか、業務を点検しなくちゃ!

## 1-3 生産性意識を高めよう

### 学習のポイント

- 生産性とは、投入量に対する産出量の大きさを表すものであり、効率といい換えることもできる。
- 製造業では製品や原材料の歩留まり、人の作業効率、機械や設備の稼働率に関する生産性指標が用いられる。
- 小売業では商品、人、スペースに関する生産性指標が用いられる。

### 1 生産性の指標を理解する

この節では、利益意識、コスト意識をもう一歩発展させ、生産性について考えてみましょう。利益とコストの双方を意識することは、すなわち生産性を高めるということになります。

生産性とは、投入量（インプット）に対する産出量（アウトプット）の大きさを表すものです。つまり、投入量に対して、どのくらいの効率で産出したかを示すものであり、「効率性」といい換えることもできます。

```text
生産性（効率） = 産出量（アウトプット） / 投入量（インプット）
```

インプットの項目には、人や製品、材料、機械、面積等、原価を構成するさまざまな数値が用いられます。

ここでは製造業や小売業で用いられる基本的な生産性指標を説明しましょう。ここで説明する指標を参考にしながら、あなたの職場にはどのような生産性指標があり、どのように使用されているか調べてみてください。

### 2 製造業で用いられる生産性指標

製造現場で使用される生産性指標には、製品や原材料の歩留まりを示す指標、作業者の効率を表す指標、機械や設備の稼働率などがあります。

**製造現場における主な生産性指標**

1. 製品、原材料の歩留まりに関する指標

```text
歩留まり率 = 良品数 / 投入数
不良品発生率 = 不良品数 / 投入数
```

2. 人の作業効率に関する指標

```text
1時間あたり完成品数量 = 完成品数量 / 時間
作業能率 = 実際作業時間 / 標準作業時間
稼働率 = 実際作業時間 / 勤務時間
```

3. 機械、設備に関する指標

```text
設備稼働率 = 実際稼働時間 / 許容稼働時間
1台あたり生産量 = 生産量 / 設備台数
```

製造現場においては、製品や原材料の歩留まりは、原材料などの使用効率を示すものとして重要な指標です。人に関しては、作業者の担当作業に対する習熟度を表す作業効率指標があり、作業者自身の適正や能力、経験、努力の結果が大きく反映されます。また、設備稼働率は、作業者の責任もありますが、それ以上に設備上の要因や設備保全体制などの管理的要因が大きく反映されます。

注: 歩留まりとは、製造したすべての製品に占める、正常な製品の割合のことです。

### 3 小売業で用いられる生産性指標

小売業は、決まった店舗の面積（スペース）で商品を販売します。この特徴から、商品、人、スペースについての生産性指標が使用されます。

**店舗で使用される主な生産性指標**

1. 商品（在庫）に関する指標

```text
商品の生産性（交叉比率） = 粗利益 / 在庫金額
商品回転率 = 売上個数（額） / 在庫数量（額）
```

2. 人の作業効率に関する指標

```text
1人1時間あたり粗利益高 = 粗利益 / 時間
1人あたり売上高 = 売上高 / 人数（8時間で1人に換算）
1人1時間あたり陳列数量 = 総陳列数量 / 時間
```

3. スペースに関する指標

```text
スペース生産性 = 粗利益高（売上高） / 売場面積
```

店舗で商品を陳列して販売する小売業にとって、その在庫がどれだけ利益を生み出したかを表す「商品の生産性」はもっとも重要な指標です。商品の生産性は、図表1-5のように分解できます。

**図表1-5 | 商品の生産性指標（交叉比率）の分解**

```text
商品の生産性 = 粗利益 / 在庫金額
             = 粗利益 / 売上高 x 売上高 / 在庫金額
             = 粗利益率 x 商品回転率
```

商品の生産性指標は、粗利益率と商品回転率の2つに分解することができます。このことから、「交叉比率」といわれることもあります。

また、小売業では、従業員におけるパート・アルバイト社員の比率が高く、勤務時間が人によって異なります。このため、人の生産性を計算する場合には、人数よりも時間数を使用するのが一般的です。

> 小売業ではスペースについての生産性も大切なんだね!

## 1-4 経営状態を判断する数値を理解しよう

### 学習のポイント

- 計数感覚を身につけるためには、財務三表の基本的な読み方を理解しておく必要がある。
- 貸借対照表は会社の健全性を判断するものであり、資産の内容や借入金等の負債金額の大きさがわかる。
- キャッシュフロー計算書は資金の収支の状態をみるもので、事業活動によって獲得した資金や投資金額の大きさがわかる。
- 損益計算書は会社の収益力をみるもので、経営プロセスごとの利益が確認できる。

### 1 会社の経営状態を判断する3つの表

計数感覚を身につけるためには、利益意識、コスト意識、生産性意識を高めるとともに、経営状態を判断する数値についても基本的な読み方を理解しておく必要があります。

企業は、年度末の時点で、その会計年度の財政状態、経営成績を明らかにします。そのときに行う手続きを決算と呼びます。このとき作成されるのが決算書（財務諸表）です。

決算書のうち、貸借対照表、キャッシュフロー計算書、損益計算書は財務三表と呼ばれます。これらの読み方を身につければ、企業の収益力や健全性、あるいは資金の動き方など、経営状態を総合的に判断することができます。

この節では、財務三表からなにが読みとれるか、その概要を学習しましょう。

### 2 貸借対照表で会社の健全性を判断する

貸借対照表には、左側（借方）に資産、右側（貸方）に負債と純資産が掲載されています。

右側は経営に必要な資金の調達方法を表しており、どれだけ負債に依存しているか否かがわかります。これによって企業の健全性を評価することができます。

資産に対する純資産の比率を自己資本比率と呼び、企業の健全性を評価するうえでもっとも重要な指標です。

```text
自己資本比率（%） = 純資産 / 資産合計 x 100
```

図表1-6の貸借対照表の例では、自己資本比率は純資産（1,200百万円）÷資産合計（2,000百万円）で60%となっています。

| 流動資産 | 売掛金 | 300 | 流動負債 | 未払費用 | 100 |
| 流動資産 | 棚卸資産 | 250 | 流動負債 | 短期借入金 | 150 |

負債は期日が来たら返済しなければならず、金利の支払いも必要になります。一方、純資産は主に株主から調達した資金を意味し、経営が継続する限り返済義務のない調達資金です。このことから、負債が多いと会社の健全性は危うくなり、純資産が多いほど健全だといえます。

| 危険な会社 | 資産に対して負債が大きく、純資産が薄い | 自己資本比率が低く、資金返済や金利負担のリスクが高い |

|  | 減価償却費 | 120 |

|  | 棚卸資産の減少 | 60 |
|  | 仕入債務の増加 | 30 |
|  | 営業活動によるキャッシュフロー | 320 |
| II 投資活動によるキャッシュフロー | 有形固定資産の取得による支出 | -180 |
|  | 投資有価証券の取得による支出 | -40 |
|  | 投資活動によるキャッシュフロー | -220 |

図表1-8を見ると、C社は営業活動によって320百万円の資金を獲得しています。これを営業キャッシュフローと呼びます。

また、C社はこの決算期に有形固定資産、投資有価証券への支出で220百万円の支出を行いました。この合計を投資キャッシュフローといいます。

さらに、短期借入金の返済や社債の発行、配当金の支払いなどの財務活動で、70百万円の資金を充当し、今期は30百万円の資金が増加しました。

C社の場合、事業活動で獲得した資金320百万円の範囲内の220百万円で投資活動を行っており、2つのキャッシュフローを差し引きすると100百万円の余りが生じています。このように営業活動によるキャッシュフローと投資活動によるキャッシュフローを合わせたものをフリーキャッシュフローといい、これがプラスであれば、健全な経営を行っているということができます。

**図表1-9 | キャッシュフロー表の3つのパターン**（単位: 百万円）

| 区分 | C社 | D社 | E社 |
|---|---:|---:|---:|
| 営業キャッシュフロー | 320 | 320 | -50 |
| 投資キャッシュフロー | -220 | -500 | -100 |
| フリーキャッシュフロー | 100 | -180 | -150 |
| 経営パターン | 健全経営型 | 積極経営型 | 不振経営型 |

図表1-9のD社は、営業キャッシュフローの金額を上回る積極的な投資活動を行っており、このためフリーキャッシュフローはマイナスになっています。この場合、不足する資金を借金か手持ちの資金の取り崩しでまかなわなければなりません。積極的な経営といえるものの、反面では経営のリスクは高まっているといえます。

また、E社の場合は、営業キャッシュフローがマイナスであり、事業活動で資金を獲得することができていません。これは、不振企業の典型的なパターンです。経営を維持するためには最低限の投資も必要であり、このままいくとどんどん借金が増加してしまいそうです。

なお、上記のようなキャッシュフロー計算書は、現在、グループ会社全体で決算を行う連結財務諸表として公表されています。

### 4 損益計算書によって会社の収益力がわかる

損益計算書は会社の収益力、つまりその会社にどれだけ利益をあげる力があるかを表したものです。このため、ただ収益と費用を集計するだけでなく、「売価と原価の差額で得た利益」「本業で得た利益」など、経営のプロセスごとにいくらの利益を獲得しているか詳しく把握できるようになっています。

外部に公表される損益計算書は、経営のプロセスごとの利益を把握できるよう、つぎの5つの段階ごとに利益を計算します。

**損益計算書で確認できる5つの利益**

| No. | 利益 | 内容 |
|---|---|---|
| 1 | 売上総利益 | 売価と売上原価の差額。粗利益（または荒利益）とも呼ばれる |
| 2 | 営業利益 | 売上総利益から、販売費、本社部門等の管理費（一般管理費）を差し引いたもの。本業で獲得した利益を意味する |
| 3 | 経常利益 | 営業利益に営業外収益を足し、営業外費用を差し引いたもの。その期の経営努力の成果を表す |
| 4 | 税引前当期純利益 | 経常利益に特別利益を足し、特別損失を差し引いたもの。その期に獲得した利益を表す。連結の場合は税金等調整前当期純利益という |
| 5 | 当期純利益 | 法人税、住民税、事業税等の税金を支払った後の利益。最終利益、配当可能利益ともいわれる |

売上原価は、販売された製品・商品の原価のことです。製造業であれば、工場で発生した材料費、人件費、経費はすべて原価となります。小売業等では、商品の仕入価格が原価になります。

営業利益は会社の本業で獲得した利益を意味します。「営業利益」と聞いたら、頭のなかで自動的に「本業の利益」と置き換えられるように覚えておきましょう。

経常利益には本業分と本業以外の両方の収益と費用が含まれていますが、いずれもその期の経営努力によって獲得したものです。このため、経常利益は、その期の経営努力の成果を表しているといえます。

特別利益と特別損失はその期の経営努力に起因するものではありませんが、その期に発生した利益と損失です。したがって、税引前当期純利益はその期の経営努力による否かに関わらず、その期に獲得した利益を表します。

なお、大企業の場合、税引前当期純利益に対して通常約30%の法人税等が課せられます。これを実効税率または税負担率といいます。

**図表1-10 | 損益計算書**（単位: 百万円）

| 区分 | 科目 | 金額 |
|---|---|---:|
| I | 売上高 | 3,000 |
| II | 売上原価 | 2,200 |
|  | 売上総利益 | 800 |
| III | 販売費・一般管理費 | 560 |
|  | 営業利益 | 240 |
| IV | 営業外収益 | 60 |
| V | 営業外費用 | 40 |
|  | 経常利益 | 260 |
| VI | 特別利益 | 70 |
| VII | 特別損失 | 80 |
|  | 税引前当期純利益 | 250 |
|  | 法人税等 | 75 |
|  | 当期純利益 | 175 |

**図表1-10の読み取り**

| 項目 | 読み取り |
|---|---|
| 売上総利益 | 売価と原価の差額分。通常、粗利益（または荒利益）といわれる |
| 販売費・一般管理費 | 店舗や本社、物流センターで発生した人件費、販売促進費、設備費等が含まれる |
| 営業外収益・営業外費用 | 営業外収益には本業以外によって獲得した収益（受取利息や受取配当金、有価証券の売却益等）、営業外費用には本業以外の費用（支払利息、有価証券の売却損等）が含まれる |
| 特別利益・特別損失 | 特別利益には土地や建物の売却益、特別損失にはその逆の売却損、除却損、リストラ費用も含まれる |
| 当期純利益 | 法人税等を支払って残った利益。最終利益、配当可能利益ともいわれる |

### 5 部門（店舗）の損益計算をみてみよう

損益計算書は、外部に公表するもののほかに、会社の内部管理用のものも作成されています。

会社全体の損益計算書は外部に公表されるものですから、どんな企業でもほとんど共通の様式で作成されます。しかし、会社の内部で使用される損益計算書は主に部門（または店舗）の管理を目的とするものですので、それぞれの会社によって様式が異なります。ここでは、一般に多く利用されている様式を例にとって説明します。

まず、会社全体の損益計算書と異なる点は、多くの場合部門の責任を明確にするため、部門貢献利益という利益が表示されていることです。売上高から売上原価を差し引いて粗利益を算出し、粗利益から部門（あるいは店舗）で発生する経費を差し引いたものが部門貢献利益です。

**図表1-11 | 部門の損益計算書**（単位: 百万円）

| 科目 | A部門 | B部門 | C部門 | 計 |
|---|---:|---:|---:|---:|
| 売上高 | 400 | 600 | 360 | 1,360 |
| 売上原価 | 300 | 460 | 300 | 1,060 |
| 粗利益 | 100 | 140 | 60 | 300 |
| 部門経費: 販売促進費 | 24 | 28 | 30 | 82 |
| 部門経費: 人件費 | 26 | 32 | 24 | 82 |
| 部門経費: 設備費 | 16 | 26 | 12 | 54 |
| 部門経費: 管理費 | 6 | 10 | 8 | 24 |
| 部門経費合計 | 72 | 96 | 74 | 242 |
| 部門貢献利益 | 28 | 44 | -14 | 58 |
| 間接経費配賦額 | 10 | 16 | 14 | 40 |
| 部門利益 | 18 | 28 | -28 | 18 |

部門の責任者は部門貢献利益について目標をもっており、これを達成することが最大の課題です。部門のメンバーは、その目標と達成状況を常に意識して行動する必要があります。

さらに、管理部門などで発生する経費を間接経費配賦額として部門で負担し、最終的な部門利益を算出します。

> あなたの部門の部門貢献利益を確認し、書き留めておきましょう。

※間接経費配賦額: どの部門にどれだけかかったかを明確に分けられない費用について、一定の基準を作成して割り振った額。

### 1-1 利益意識を高めよう

- 企業の最終的な目的は「利益をあげること」である。利益は、収益 - 費用 = 利益で求められる。
    
- 収益が費用を上回れば利益がでて、収益よりも費用が上回れば損失がでる。
    

### 1-2 コスト意識を高めよう

- コスト意識を身につけるためには、まずは自分にかかるコストを意識するとよい。
    
- 在庫を過剰にもつと、①金利費用、②保管費用、管理費用、③在庫ロス、④値引き、廃棄ロス、⑤経営資金の圧迫といった経営に対する悪影響がある。
    
- コスト意識が高まると、ふだんの業務でムリ、ムダ、ムラを発見できるようになる。
    

### 1-3 生産性意識を高めよう

- 生産性とは、投入量(インプット) に対する産出量(アウトプット)の大きさを表すものである。
    
- 製造業では、製品、原材料の歩留まりに関する指標、人の作業効率に関する指標、機械、設備に関する指標を用いる。また、小売業では、商品(在庫)に関する指標、人の作業効率に関する指標、スペースに関する指標を用いる。
    

### 1-4 経営状態を判断する数値を理解しよう

- 決算書のうち貸借対照表、損益計算書、キャッシュフロー計算書のことを財務三表と呼ぶ。
    
- 貸借対照表からは健全性、キャッシュフロー計算書からは資金の流れ、損益計算書からは収益力を読み取ることができる。

また、製造現場で用いられている標準原価の考え方を理解し、実際に発生した原価との差異を分析する方法やコストダウンの進め方を学習します。

| 2-1 | 原価の構成を理解しよう |

## 2-1 原価の構成を理解しよう

### 学習のポイント

- 原価計算は、①正しい決算書を作成する、②販売価格を決定する、③原価の管理を行う、④採算計算を行う、という4点の目的で行う。
- 製造原価は、①材料費、②労務費、③経費の3つに分類される。

### 1 なぜ原価計算を行うのか

原価計算とは、文字どおり、製品の原価を計算することです。ではなぜ、製品の原価を計算しなければならないのでしょうか。

原価計算を行う目的は、つぎの4点があげられます。

**図表2-1 | 原価計算の目的**

| 目的 | 内容 |
|---|---|
| 1 | 正しい決算書を作成する |
| 2 | 販売価格を決定する |
| 3 | 原価の管理を行う |
| 4 | 採算計算を行う |

#### ①正しい決算書を作成する

企業は、取引先や金融機関、株主などの利害関係者に対して経営状況を定期的に報告しなければなりません。この報告では、損益に関して正確な情報を開示することがもっとも重要です。

このとき、製品の原価計算がいいかげんに行われていると、損益の計算も正確ではなくなってしまいます。

正しい決算書を作成するためには、正しい原価計算が前提となるのです。

#### ②販売価格を決定する

製品の価格をいくらに設定するかということは、企業にとってもっとも重要な決定事項の1つです。高すぎれば売れず、低すぎれば利益が出ません。

販売価格を決定する際には、お客さまの感じる価値に対して適正な価格を設定すると同時に、いくらの利益をあげることができるかを考慮することも重要です。正確な原価が計算できなければ、価格の決定に際して、的確な判断ができなくなってしまいます。

#### ③原価の管理を行う

企業が利益を獲得し続けるためには、常に経営努力が必要です。この経営努力のうち、重要なものの1つにコストダウンがあります。

コストダウンを実現するためには、製品の製造にかかった原価が予定した原価（標準原価）の範囲に収まったのかどうかを常にチェックしていなければなりません。

もし、予定した原価をオーバーしているとしたら、その原因を徹底的に究明しなければ、永久にコストダウンは実現しません。

#### ④採算計算を行う

特別注文がある場合や、新しく設備を導入する計画がある場合など、経営の意思決定が必要な場面では、採算が取れるかどうかを判断しなければなりません。

お客さまの要求価格で受注して利益が出るのか、あるいは新しい設備の導入で原価がどの程度安くなるのかといった採算計算には、正確な原価の計算が大変重要な役割を果たします。

> コストダウンや経営の意思決定にも使われるんだ。

### 2 企業の活動プロセスと費用

具体的な原価計算の学習に入る前に、企業の活動プロセスを確認しておきましょう。

費用は、企業の活動プロセスの各段階で発生します。企業の活動プロセスは業種によって異なりますので、ここでは、製造業、販売業、サービス業の3つの業種ごとに、活動プロセスと、そこで発生する費用をみていきましょう。

#### ①製造業の活動プロセスと費用

製造業には、一般に製品開発、購買、製造、物流、販売、全般管理という活動プロセスがあり、それぞれで費用が発生します。

**図表2-2 | 製造業の活動プロセス**

| 活動プロセス | 主な場所 | 損益計算上の扱い |
|---|---|---|
| 製品開発 | 本社 | 販売費・一般管理費 |
| 全般管理 | 本社 | 販売費・一般管理費 |
| 購買 | 工場 | 製品の製造原価 |
| 製造 | 工場 | 製品の製造原価 |
| 物流 | 物流センター | 販売費・一般管理費 |
| 販売 | 営業所 | 販売費・一般管理費 |

各活動が行われる場所は、製品開発や全般管理は本社で、購買、製造は工場だとします。また、物流業務や販売活動は物流センターや営業所で行われているとします。

このうち、本社で発生した開発費や全般管理費、物流センターや営業所で発生した物流費や販売費は販売費・一般管理費として分類され、会社の損益計算書に計上されることになります。

しかし、工場で発生した購買費用や、製造にかかわる費用は、すべて製品の製造原価として集計されます。

#### ②販売業の活動プロセスと費用

小売業や商社などの販売業には、購買、物流、販売、全般管理といった活動のプロセスがあります。

**図表2-3 | 販売業の活動プロセス**

| 活動プロセス | 主な場所 | 損益計算上の扱い |
|---|---|---|
| 購買 | 本社 | 商品の購入代金は仕入原価 |
| 全般管理 | 本社 | 販売費・一般管理費 |
| 物流 | 物流センター | 販売費・一般管理費 |
| 販売 | 店舗 | 販売費・一般管理費 |

このうち、購買段階での商品購入代金は仕入原価となります。物流、販売で発生する費用は、販売費・一般管理費です。また、企業の全般管理に関して発生した費用も販売費・一般管理費です。

#### ③サービス業の活動プロセスと費用

ひと言でサービス業といってもさまざまな業種、業態があります。ここでは、外食産業とソフトウェア業を例にとって説明しましょう。

外食産業では、通常、食材費のみを原価とし、それ以外の費用を販売費としますが、客席と厨房をしっかり分けて管理している場合、厨房で発生した食材費、人件費、光熱費などは原価となります。これに対し、客席などで発生する費用は、販売費とします。

ソフトウェア業では、システム開発が長期に及び、そこで多くのコストが発生するのが特徴です。このような場合、開発中に発生した人件費や外注費などの費用は原価として集計し、システムが完成し顧客に引き渡された時点で売上原価となります。

もし完成しないまま決算を迎えると、在庫として資産に計上されることになります。

### 3 原価とは何か

ここまで、それぞれの費用がどの段階で発生するのかを確認してきました。費用のうち、通常もっとも大きな割合を占めるのは原価です。原価についてもう少し詳しく学習しましょう。

#### 1 さまざまな原価

「原価」と呼ばれるものには、いくつか種類があります。たとえば、製造業の「製造」のプロセスでかかる費用、つまり原料や材料を仕入れて製造するまでの活動で発生する原価は製造原価です。また、販売業の「購買」のプロセスで発生する、商品を仕入れるための原価は仕入原価と呼ばれます。

さらに、製造原価や仕入原価と販売費・一般管理費を合計したものを、一般に総原価と呼びます。総原価は、経営活動を行うために消費したすべての費用の合計です。

なお、総原価は「経営活動を行うために消費したすべての費用」と説明しましたが、つぎのような項目は原価に含めません。原価に含まない費用のため、非原価項目と呼ばれています。

**非原価項目の内容**

| 区分 | 内容 |
|---|---|
| 主な事業に関連しない項目 | 本業外の費用、損失（支払利息、投資損失など） |
| 経営努力以外の損失 | 事故による損失、火災・風水害による損失 |
| 税法上の特別措置による損金項目 | 特別償却費など |
| 利益処分項目 | 法人税、住民税、役員賞与、配当金など |

ここまで説明してきたとおり、業種や業態により、原価の構成は異なります。しかし、原価計算といえば、一般に製造原価を求めることを指します。ここからは、製造原価の計算について解説していきます。

#### 2 製造原価の3要素

製造原価は、材料費、労務費、経費の3つに分類されます。これらを原価の3要素と呼びます。

**図表2-4 | 原価の構成**

| 構成 | 内容 |
|---|---|
| 販売価格（売上高） | 総原価と利益で構成される |
| 総原価 | 製造原価と販売費・一般管理費で構成される |
| 製造原価 | 材料費、労務費、経費で構成される |

では、原価の3要素の内容を確認していきましょう。

##### 1 材料費

材料費は、原料や材料を使うことによってかかる費用です。主なものは、主要材料費（素材費とも呼びます）と、買入部品費です。主要材料費は、製品の本体を構成する物品の購入費です。また、買入部品費は外部から購入してそのまま製品の一部に組み込む部品の購入費です。このほかに、燃料費、補助材料費（溶接材、補修用鋼材、テープ、ネジなど）、消耗工具備品費（機械油、布、スパナ、ドライバーなど）があります。

さらに、外部副費と呼ばれる買入手数料、引取運賃、関税、荷役費などや、内部副費と呼ばれる購入事務費、検収費、保管費なども材料費に含まれます。

**材料費に含まれるもの**

- 主要材料費（素材費）
- 買入部品費
- 燃料費
- 補助材料費
- 消耗工具備品費

##### 2 労務費

労務費は、労働力に支払われる対価です。賃金、給料、雑給、従業員賞与手当、退職給与引当金繰入額、福利費があります。

工場で製品を製造する人に支払われる給与は賃金、工場の事務部門や技術部門で働く人に支払われる給与は給料です。また、雑給とは、工場の臨時雇いの人やパートタイマーに支払われる給与のことです。ボーナスや通勤手当、食事補助などの各種手当は従業員賞与手当、社会保険料は福利費となります。

**労務費に含まれるもの**

- 賃金
- 給料
- 雑給
- 従業員賞与
- 手当
- 退職給与引当金繰入額
- 福利費

##### 3 製造経費

製造経費は、材料費、労務費に該当しない費用の総称です。製造経費は、金額を把握する方法によってつぎのように分類されます。

**図表2-5 | 製造経費の内容**

| 分類 | 費用の種類 |
|---|---|
| 支払経費 | 外注加工費、特許権使用料、修繕費、旅費交通費、事務用品費など |
| 月割経費 | 賃借料、減価償却費、火災保険料、固定資産税など |
| 測定経費 | 電力料、ガス代、水道代など |
| 発生経費 | 棚卸減耗費、仕損費 |

支払経費は、支払いを行ったり、請求されたりした時点で金額を把握する経費です。外注加工費や修繕費、旅費交通費などがあります。

月割経費は、年や月の金額が固定で決まっている経費です。代表的なものは、賃借料や減価償却費です。

測定経費は、メーターで測定して使用金額を把握する経費です。電気代やガス代などがあります。

発生経費は、実際にその事実が発生してはじめて把握できる経費です。主なものに、棚卸減耗費や仕損費などがあります。

##### 4 販売費・一般管理費の内容

販売費・一般管理費は工場（製造部門）以外で発生する費用であり、販売活動や配送業務、管理業務に関する費用です。

ここで注意しなければならないのは、人件費や配送業務、管理業務に関する費用だったとしても、それが工場で発生した場合は製造原価（製造経費）になるということです。人件費でも本社や営業部で発生した場合にのみ、販売費・一般管理費になります。

なお、販売費・一般管理費は、図表2-6のように大きく分類すると販売促進費、人件費、設備費などに分けられます。しかし、細かく内容を分けて表示する会社はほとんどなく、多くの会社はまとめて販売費・一般管理費として表示しています。

**図表2-6 | 販売費・一般管理費の内容**

| 分類 | 費用の種類 |
|---|---|
| 販売促進に関する費用 | 広告宣伝費、販促費、支払手数料、交際費、配送費、保管費など |
| 人に関する費用 | 給与・賞与、残業代、パート・アルバイト代、通勤費、福利厚生費など |
| 設備に関する費用 | 賃借料、減価償却費、修繕費、保険料、租税公課、備品費など |
| その他の費用 | 旅費交通費、水道光熱費、事務消耗品費、通信費、会議費など |

※1 減価償却費: 固定資産を購入した費用を、その耐用年数で分割した費用。詳しくは3-3-2で学習します。

※2 棚卸減耗費: 棚卸を行なった際に、理論上の在庫数量と実際の在庫数量との間に発生する差によって生じる費用。

※3 仕損費: 製品の製造中に生じた失敗による費用。

## 2-2 原価計算のしくみを理解しよう

### 学習のポイント

- 原価計算には、個別原価計算と総合原価計算の2つの方法がある。
- 仕掛品がある場合の原価計算は、材料費と加工費に分けて行う。
- 材料費には、平均法や先入先出法といった複数の計算方法がある。

### 1 総合原価計算のしくみ

製品の製造方法は、顧客の注文にもとづいて一品一品異なる製品を製造する個別受注生産方式と、市場での販売を見込んで同じ種類の製品を継続して作り続ける見込生産方式があります。

個別受注生産方式で製造が行われる場合には、個別原価計算という計算方法が、見込生産方式の場合には総合原価計算という計算方法が適用されます。

ここでは、総合原価計算について、簡単な例から説明することにします。

#### 1 簡単な原価の計算をしてみよう

まず、つぎのような簡単なケースで製造原価の計算をしてみましょう。

**【ケース1】** 今月、X工場では製品Aの製造を行い、発生した費用と完成品数量はつぎのとおりです。製品Aの製造原価はいくらになるでしょうか。

■費用（単位: 千円）

| 費目 | 金額 |
|---|---:|
| 材料費 | 500 |
| 労務費 | 200 |
| 経費 | 300 |
| 計 | 1,000 |

■完成品数量: 1,000個

このケースでは、1,000千円の費用を投入して、製品Aを1,000個製造しています。製品Aの製造原価はつぎのように計算します。

```text
製造原価 = 投入コスト（1,000千円） ÷ 完成品数量（1,000個） = 1,000円
```

#### 2 複数製品を製造する場合の原価計算

さて、今度は少し複雑になります。つぎのケースを考えてみましょう。

**【ケース2】** X工場では、業務を拡大するために、製品A以外に、製品Aとほとんど同じで、一部分の素材を変えた製品Bも製造することになりました。この業務拡大により変化した費用と完成品数量は、つぎのとおりです。製品Bの製造原価はいくらになるでしょうか。

■当初の費用（単位: 千円）

| 費目 | 金額 |
|---|---:|
| 材料費 | 500 |
| 労務費 | 200 |
| 経費 | 300 |
| 計 | 1,000 |

■当初の完成品数量: 製品A 1,000個

■業務拡大後の費用（単位: 千円）

| 費目 | 金額 |
|---|---:|
| 材料費 | 800 |
| 労務費 | 250 |
| 経費 | 450 |
| 計 | 1,500 |

■業務拡大後の完成品数量

| 製品 | 完成品数量 |
|---|---:|
| 製品A | 1,000個 |
| 製品B | 800個 |

単純に計算すると、つぎのようになります。

```text
製品Aの原価 = 費用（1,000千円） ÷ 完成品数量（1,000個） = 1,000円
製品Bの原価 = 費用（1,500千円 - 1,000千円） ÷ 完成品数量（800個） = 625円
```

この計算では、製品Aの材料費、労務費、経費はそのままです。一方、製品Bの原価は材料費を（800千円 - 500千円）で300千円、労務費を（250千円 - 200千円）で50千円、経費も（450千円 - 300千円）で150千円としています。

はたして、本当にこれでよいでしょうか。少し考えてみましょう。製品Bは、製品Aとほとんどの部分で同じ材料を使います。また、製品Bの経費は業務拡大で増えた分のみとしてよいでしょうか。さらに労務費も、業務拡大により増やした従業員が製品Aも作る可能性を考えなくてよいでしょうか。

このように考えていくと、原価のなかで、「この部分は製品A、この部分は製品B」とはっきり割り振れないものがありそうです。この点が考慮されていないため、先ほどの計算は正しいとはいえません。

複数の製品を製造している場合には、特定の製品の製造だけに発生したと判別できる費用と、どの製品の製造のために発生したか判別できない費用の2つに分けて計算します。前者を製造直接費、後者を製造間接費と呼びます。

たとえば、材料費のうち特定の製品を製造するために購入したものは製造直接費です。また、特定の製品の製造にしかかかわらないような作業者がいれば、その労務費も製造直接費となります。

これに対し、工場の管理部門の経費や建物・機械の減価償却費、火災保険料などは、どの製品にいくらかかったか判別しにくく、製造間接費となります。

原価の3要素である材料費、労務費、経費という分類は形態別分類といわれるものです。これに対し、原価の計算を行う場合に、製品別に直接集計できるかできないかで分類したものが、直接費、間接費です。これは、製品別分類と呼ばれます。

**図表2-7 | 直接費・間接費の分類**

| 区分 | 製品原価への入れ方 |
|---|---|
| 製造直接費 | 製品ごとに直接集計する。材料費・労務費・経費のうち、製品Aや製品Bに直接結びつく部分をそれぞれの製品原価に入れる。 |
| 製造間接費 | どの製品にいくらかかったか直接判別できないため、いったん製造間接費として集計し、一定の基準で各製品へ配賦する。 |

#### 2 製造間接費を製品原価に配分するには

製造直接費は製品ごとに集計することができますが、製造間接費はどのように計算するのでしょうか。

製造間接費は、一定の基準を作成して割り振ることになります。これを配賦といい、配賦のための基準を配賦基準と呼びます。

配賦基準は、それぞれの製造間接費が各製品の製造にどのようにかかわっているか、できるだけ因果関係のある数値を使って決定します。図表2-8は、その例です。

**図表2-8 | 製造間接費の配賦基準の例**

| 製造間接費の種類 | 配賦基準に使用する数値 |
|---|---|
| 間接労務費 | 作業時間 |
| 動力費 | 機械稼働時間 |
| 機械の減価償却費 | 機械稼働時間 |
| 建物保険料、土地賃借料 | 所要面積比 |
| 購買部門費 | 材料の払出数量（金額） |
| 工場管理部門費 | 配置人員数 |

## 第2章 まとめ

### 2-1 原価の構成を理解しよう

- 原価計算は、①正しい決算書を作成する、②販売価格を決定する、③原価の管理を行う、④採算計算を行うという4つの目的で行われる。
    
- 製造原価は、材料費、労務費、経費の3つに分類され、これを原価の3要素と呼ぶ。
    

### 2-2 原価計算のしくみを理解しよう

- 実際原価計算とは実際にかかった原価を集計する方法で、同じ種類の製品を継続して作り続ける場合の総合原価計算と、異なる製品を個別に受注して製造する場合の個別原価計算がある。
    
- 複数の製品を製造している場合は、かかった費用について、どの製品のために発生したかが明らかな製造直接費と、明らかでない製造間接費に分けて考える必要がある。
    
- 仕掛品がある場合の原価計算では、完成品原価と仕掛品原価に分けて計算する。このうち仕掛品原価は、完成までの進捗率をもとに完成品に換算して計算する。
    
- 材料費の計算には、移動平均法、総平均法、先入先出法など、複数の方法がある。
    

### 2-3 標準原価計算とは

- 標準原価計算を行う目的のうちもっとも重要なものは、目標とする原価との差異を分析し、コストダウンにつなげることである。
    
- 標準原価の差異分析は、直接材料費、直接労務費、製造間接費の3つに分けて行う。

たとえば、工場の総務・経理部門の社員や守衛の人件費も製造間接費です。このような費用を製品に正しく配分するためには、どのような配賦基準にすればよいでしょうか。

これまで学習してきたケースは、月の終わりに製品がすべて完成している例でした。しかし、実際の製造現場ではそのようなことはまれであり、月末時点では製造途中で未完成の製品が残っている場合がほとんどです。

製造途中の作りかけの製品は、仕掛品(しかかりひん)と呼びます。

今度は、月末に仕掛品がある場合の製品原価の計算についてみていきましょう。

**【ケース4】** 今月、X工場では製品Aの製造を行い、発生した費用と完成品数量、および仕掛品の進捗状況はつぎのとおりでした。製品Aの製造原価はいくらになるでしょうか。

■費用 (単位:円)

|費目|金額|
|---|---|
|材料費|500,000|
|労務費|200,000|
|経費|300,000|
|加工費小計|500,000|
|計|1,000,000|

■完成品数量: 900個

■仕掛品の状況

|項目|材料費|加工費|数量|備考|
|---|---|---|---|---|
|月初|39,000|30,000|100個|加工の進捗率50%|
|月末|計算対象|計算対象|200個|加工の進捗率50%|

仕掛品がある場合には、はじめに、費用と仕掛品の状況を、「材料費」と「加工費」に分類します。加工費とは、労務費と経費を合計したものです。これは、一般に製造工程のはじめの段階で投入される材料費と、製造工程の進捗に応じて消費される労務費や経費とでは、計算方法が異なるためです。

### ①材料費の計算

月初にあった仕掛品の材料費は39,000円です。それに加え、今月新たに500,000円投入しました。したがって材料費の総額は、月初仕掛品(39,000円)＋今月投入分(500,000円)＝539,000円となります。

これに対して、月末の完成品数量は900個、仕掛品数量は200個です。つまり、アウトプットの総量は900個＋200個＝1,100個となります。

これを図で表したものが、つぎの図表2-12です。

**図表2-12 | 材料費のインプットとアウトプットの関係**

- **インプット総額: 539,000円**

    - 今月投入分 (数量1,000個): 500,000円
        
- **アウトプット数量: 1,100個**
    
    - 完成品数量: 900個

完成品数量は、アウトプット数量1,100個のうちの900個ですから、原価の計算はつぎのように行います。

$$完成品原価 = インプット総額(539,000円) \times \frac{900個}{1,100個} = 441,000円$$

また、仕掛品の原価はつぎのようになります。

$$仕掛品原価 = インプット総額(539,000円) \times \frac{200個}{1,100個} = 98,000円$$

### ②加工費の計算

労務費や経費などの加工費は、材料費と異なり製造プロセスの進捗状況に応じて発生します。

月初や月末に仕掛品がある場合、その加工費はまず、進捗率という数値を用いて完成品にすると何個分になるのかを換算します。たとえば、材料ベースで仕掛品が100個あり、その仕掛品の完成までの進捗状況が60%だとします。この場合、加工費については完成品60個分として換算するのです。

では、ケース4について完成品と仕掛品の原価を計算してみましょう。

月初にあった仕掛品の加工費は30,000円です。それに加え、今月新たに500,000円かかりました。したがって加工費の総額は、月初の仕掛品(30,000円)＋今月投入分(500,000円)＝530,000円となります。

これに対し、月末の仕掛品の数量は材料ベースで200個、進捗率が50%であるため、完成品に換算した場合のアウトプットは仕掛品数量(200個)×進捗率(50%)＝100個となります。また、完成品数量は900個ですから、アウトプットの合計は1,000個です。

## 4 材料費の計算のしくみ

先ほどのY工場のケースでは、材料費について、どの時点で完成したものかによらず、すべて同じ額がかかるものとして計算しました。しかし、材料費は、価格変動や購入ロット数などの条件によって購入単価が異なります。倉庫に同じ材料が保管されていても、一つひとつをみれば購入時の単価は異なっている場合がほとんどでしょう。

商店では、倉庫にある商品を少しずつ出庫し、店の棚に並べて販売します。その際、先に仕入れたものから順に出庫していくのが普通です。古いものから順に販売していくのが、一般的な「物の流れ」です。

しかし、企業でコスト計算を行う場合には、この流れに固執する必要はなく、いくつかの計算方法が認められています。そのうち、代表的なものは平均法と先入先出法です。

### ①平均法

購入した材料のそれぞれの単価を平均して払出単価を決定する方法です。新しく材料を購入するたびに、それまであった材料単価と平均していく移動平均法と、1カ月間に購入した材料の平均単価を割り出す総平均法があります。

### ②先入先出法

先に購入した材料から先に払出すという考え方で払出単価を計算する方法です。先にあった材料の単価から順次払出単価を決定していきます。

## 2-3 標準原価計算とは

**学習のポイント**

- 標準原価計算を行うもっとも重要な目的は、コストダウンにつながる原価管理を行うことである。
- 標準原価と実際原価との差異分析は、材料費、労務費、製造間接費ごとに行い、改善のための対策をしっかり実行することが大切である。

## 1 標準原価計算とその目的

### ①標準原価計算とは

製品の原価を計算する場合、多くの企業では標準原価計算のしくみを導入しています。

標準原価計算とは、想定された環境のもとで製造する場合の製品1個あたりの目標原価をあらかじめ設定し、製品が完成した段階で完成数量に目標原価を乗じて製品製造原価とする方法です。

目標とする原価のことを標準原価といい、材料費、労務費、経費ごとに標準原価を決定します。製品が完成した時点で、つぎのように製品製造原価を計算します。

$$(製品1個あたり) 標準原価 \times 生産数量 = 製品製造原価$$

製品の原価を把握するには時間がかかります。材料費はあとから値引き交渉があったり、電気代はメーターで確認したり、諸経費はあとから請求書が送られてきたりします。この不便を避けるために、製品1個あたりの標準原価を決定しておき、生産数量に応じていくらの原価がかかったか目安を決めてしまいます。これにより、利益の計算もすぐにできるようになります。

標準原価計算を導入するもう1つの目的は、原価の管理を行うためです。ここでいう標準原価は、効率よく製造したらこのくらいの金額でできるであろうという目標とする原価です。科学的な調査分析にもとづいて決定された合理的な原価ともいえます。

製造工程ではムリ、ムダ、ムラやアクシデントが発生し、多くの場合、実際の原価にはさまざまなロスが含まれます。そのため、「標準原価こそが正しい原価であり、実際の原価は何らかの誤りを含んだ原価である」といわれることもあります。

標準原価と実際の原価を比較し、そこに差(原価差異)があった場合、製造現場でなんらかの好ましくない事態が発生したものと推定されます。その原因を追究して、以後のコストダウンに結び付けようとすること、つまり原価管理(=原価コントロール)を行うことが標準原価計算を導入するもっとも重要なねらいなのです。

### ②標準原価計算の目的

なぜこのような原価の計算方法を行うのか、目的の1つは原価をはやく知ることができるからです。原価がいくらかかったかが明らかになるには時間がかかります。そこで製品1個あたりの標準原価を決定しておき、生産数量に応じて原価の目安を決め、利益計算をすぐに行えるようにします。

もう1つの目的は、原価管理です。標準原価を目標として設定し、実際原価との差異を分析して原因を探り、改善策につなげます。

## 2 標準原価計算による原価管理のプロセス

標準原価を用いてどのように原価管理を行うか、そのプロセスをみてみましょう。

### ①原価標準の決定

製品1個あたりの標準原価を原価標準といいます。標準原価を用いて原価管理を行うには、まず直接材料費、直接労務費、製造間接費ごとに原価標準を決定します。

この原価標準は、「理想的な製造環境のもとで、正しい手順で」製造された場合の厳しいレベルで設定しなければなりません。原価標準のレベルが低い場合、製造工程で多少ミスや事故が発生しても原価差異が発生せず、コストダウンの活動は進みにくくなってしまうからです。

### ②製造部門への連絡、製造の開始

製造現場の担当者や、原価管理の責任者に対して原価標準を報告します。このとき、金額だけでなく、そのような原価標準を設定した背景や手順、留意点なども報告し、納得してもらうことが重要です。製造現場の担当者に原価標準にむけてチャレンジする姿勢がなければ、目標の達成は難しいものです。

### ③実際原価の把握

実際にいくらの材料費や労務費、経費がかかったかを把握します。実際原価の把握はできるだけ早いタイミングで行うことが重要です。実際原価の集計が遅れると、つぎの差異分析や原因の探求が遅くなり、その間余分なコストが生じてしまうことになるからです。

### ④原価差異の分析と対策の実施

標準原価と実際の原価に差異が生じた場合、すみやかに原因を分析・探求し、対策を実施します。成果をあげるためには改善の実行力が重要です。製造現場の担当者や原価管理者には、改善に向けたリーダーシップが求められます。

### ①直接材料費の差異分析の進め方

図表2-15によれば、直接材料費には216,000円の原価差異が生じています。この原価差異について分析していきましょう。

直接材料費に原価差異が生じた場合には、まずその差異の発生原因によって価格差異と数量差異に分けます。価格差異とは購買過程で生じた差異のことです。また、数量差異は製造過程で材料を予定よりも多く使用してしまったことによる差異です。

**図表2-16 | 直接材料費の差異分析**

- グラフ概要: 縦軸は単価、横軸は使用量。標準単価300円・標準使用量6,000kgの四角形が標準直接材料費1,800,000円を表す。実際単価320円・実際使用量6,300kgとの差は、上側の価格差異と右側の数量差異に分けて示される。

図表2-16の①の部分は価格差異を表し、つぎの計算で求められます。

$$価格差異 = (実際単価 - 標準単価) \times 実際使用量$$

この計算式にあてはめると、図表2-16の価格差異はつぎのようになります。

$$価格差異 = (320円 - 300円) \times 6,300kg = 126,000円$$

価格差異は、材料を予定の購入価格よりも高く買ってしまったことが原因の原価差異です。

また、②の部分は数量差異を表し、つぎの計算で求められます。

$$数量差異 = (実際使用量 - 標準使用量) \times 標準単価$$

この計算式にあてはめると、図表2-16の数量差異はつぎのようになります。

$$数量差異 = (6,300kg - 6,000kg) \times 300円 = 90,000円$$

数量差異は、想定した材料の消費量よりも多く使用したことが原因の原価差異です。不良品が発生した、材料の使用のしかたがまずかったなど、製造過程でロスが発生したということです。詳しい原因を徹底的に追及し、同じようなロスが繰返し発生しないように対策を講じなければなりません。

### ②直接労務費の差額分析の進め方

図表2-15によれば、直接労務費には174,000円の原価差異が生じています。この原価差異について、分析していきましょう。

直接労務費は、特定の製品を製造するために行われた作業時間に賃率を乗じて計算されます。

標準と実際の原価差異が生じた場合、まず賃率差異と作業時間差異に分けます。こちらも、わかりやすくするために図で表してみましょう。

**図表2-17 | 直接労務費の差異分析**

- グラフ概要: 縦軸は賃率、横軸は作業時間。標準賃率900円・標準作業時間1,800時間の四角形が標準直接労務費1,620,000円を表す。実際賃率920円・実際作業時間1,950時間との差は、賃率差異と作業時間差異に分けて示される。

図表2-17の③の部分は賃率差異を表します。賃率差異は、つぎの計算で求められます。

$$賃率差異 = (実際賃率 - 標準賃率) \times 実際作業時間$$

この計算式にあてはめると、図表2-17の賃率差異はつぎのようになります。

$$賃率差異 = (920円 - 900円) \times 1,950時間 = 39,000円$$

賃率差異は、予定よりも高い人件費の人を使用してしまったことによるものです。これは、人の手配や採用方法の問題です。

※賃率: 時間あたりの労務費のこと。

図表2-17のもう一方の差異は、作業時間差異です。作業時間差異は、つぎの計算で求められます。

$$作業時間差異 = (実際作業時間 - 標準作業時間) \times 標準賃率$$

この計算式にあてはめると、図表2-17の作業時間差異はつぎのようになります。

$$作業時間差異 = (1,950時間 - 1,800時間) \times 900円 = 135,000円$$

作業時間差異は、製造過程で想定した作業時間よりも多く時間がかかったことによる原価差異です。手待ち時間や不良品などが発生して余分な時間がかかったということであり、製造過程の作業効率の問題として、詳しい原因を追及して対策を講じる必要があります。

### ③製造間接費の差異分析の進め方

製造間接費に原価差異が発生した場合、それがどのような原因で生じたものかという分析は、材料費や労務費に比べてやや難しくなります。

製造間接費は、もともと特定の製品の製造にいくらかかったかわかりにくく、配賦基準によって配分された費用です。そのしくみを理解し、どの部分がどの部門の責任となる原価差異なのかを明確にしなければなりません。

まず、図表2-15の例に関する製造間接費のデータ部分だけを抜き出して整理してみましょう。

**図表2-18 | 製造間接費のデータ**

| 項目 | 数値 |
|---|---:|
| 予定生産量 | 3,400個 |
| 基準操業度 | 1,700時間 |
| 標準製造間接費 | 1,500,000円 |
| 製品1個あたり基準操業度 | 0.5時間 |
| 1個あたり標準製造間接費 | 500円 |
| 実際生産量 | 3,000個 |
| 実際操業度 | 1,600時間 |
| 実際製造間接費 | 1,800,000円 |
| 標準操業度 | 1,500時間 |

基準操業度とは、予定生産量を製造する場合に予定される操業時間のことです。ここでは、予定生産量3,400個を製造するにあたって1,700時間の操業時間が見込まれています。この基準操業度を製品1個あたりに換算すると0.5時間となります。

ただし、このケースでは、実際生産量は3,000個でした。3,000個を製造した場合の標準製造間接費は1,500,000円です。標準製造間接費を製品1個あたりに換算すると500円となります。

以上のことから、予定生産量どおりに3,400個製造した場合の製造間接費は、1,700,000円が想定されたことになります。

これらの関係をグラフに表したものが、図表2-19です。

**図表2-19 | 製造間接費の原価差異分析**

- グラフ概要: 縦軸は製造間接費、横軸は操業度。標準製造間接費1,500千円、予定製造間接費1,700千円、実際製造間接費1,800千円を示し、横軸には標準操業度1,500時間、実際操業度1,600時間、基準操業度1,700時間を置く。差額300千円は、能率差異100千円、操業度差異100千円、予算差異100千円に分解される。

このケースでは、生産量調整の指示があり、実際には3,000個しか製造しませんでした。このため、標準操業度は1,500時間、標準製造間接費は1,500,000円となりました。

これに対し、実際の製造間接費は1,800,000円発生しており、この差額300,000円が原価差異です。

**①能率差異**

実際の生産量は3,000個でしたから、本来は1,500時間で生産すべきでした。しかし、実際の操業時間は1,600時間です。つまり、100時間余分にかかったことになります。このような原価差異を能率差異といいます。能率差異は、作業の効率が悪かったことによる原価差異です。

**②操業度差異**

操業時間は1,700時間を予定していましたが、実際には1,600時間でした。この差によって生じた100,000円の原価差異が操業度差異です。操業度差異は、生産量調整の指示があったために生じたものです。

**③予算差異**

予定されていた製造間接費1,700,000円に対して、実際には1,800,000円が発生しました。これは予算の見積り方法により生じた原価差異であり、予算差異といわれます。

製造間接費については、現場の製造部門にとっては能率差異の原因を重点的に分析する必要があります。

# 第3章 採算計算の進め方を理解しよう

## 3-1 損益分岐点はいくらか？

**学習のポイント**

- コストには、活動に応じて増減する変動費と変化しない固定費がある。
- 会社や部門の経営には、ある一定の売上を上げると赤字から黒字に転換する損益分岐点がある。
- 会社や部門経営の安全性を判断する指標として、損益分岐点比率や安全余裕率がある。

### 1 損益分岐点は黒字と赤字の境

企業活動の目的は、利益を得ることです。この目的を達成する具体的な目標を立てるためには、黒字と赤字の境となる売上高、つまり、売上と費用がちょうど一致する売上高を把握することが重要です。このような、売上と費用が一致する売上高を損益分岐点と呼びます。

この損益分岐点を把握できれば、目標とする利益を得るためにはあといくら売上を伸ばせばよいのか、あるいは、あといくら売上高が減少すると赤字となってしまうのか、ということが明確になります。

では、損益分岐点はどのように導き出すのでしょうか。実は、この計算はそう簡単ではありません。考え方をわかりやすく理解するために、身近な例で考えてみましょう。

**【ケース6】** 職場の忘年会の幹事役を任されたAさんは、会場を手配しようとB飲食店に見積りを依頼しました。以下の条件をもとに、Q1とQ2について考えてみましょう。

- 1人あたり飲み物代と料理代: 3,000円
- 席料: 50,000円

**Q1** 1人あたりの会費を5,000円とすると、赤字を出さないためには、何名の参加者を集めなくてはならないでしょうか。

**Q2** 30名の参加者が集まったとすると、いくらの余り（黒字）が出るでしょうか。

#### ①飲食店の費用の性質

まず、この忘年会にかかる費用の「席料」と「1人あたり飲み物代と食事代（飲食代）」について、それぞれの性質を確認しましょう。席料は当日の参加人数によらず50,000円、飲食代は当日の参加人数 x 3,000円です。

この関係を、縦軸を費用、横軸を参加者としてグラフに表すと、図表3-1のようになります。

**図表3-1 | 参加者と費用の関係**

- グラフ概要: 横軸は参加人数（人）、縦軸は費用（千円）。席料は参加者数にかかわらず50千円の水平線で表される。飲食代は参加者1人あたり3千円ずつ増えるため、10人で30千円、20人で60千円、30人で90千円となる。費用合計は席料50千円を起点として右肩上がりに増え、10人で80千円、20人で110千円、30人で140千円となる。

### 2 赤字を出さないためには何人集めればよいか

席料は、参加人数の増減にかかわらず50,000円ですから、横軸と平行した線になります。一方で飲食代は参加人数が1人増えるごとに3,000円増えますから、席料50,000円を起点に、右上がりの線になります。

参加人数が0人の場合には席料しか発生しないため、費用総額は50,000円です。参加人数が10人の場合は80,000円、20人では110,000円、30人では140,000円と増加していきます。

つぎに、会費収入の線を加えた図表3-2をみてみましょう。

**図表3-2 | 忘年会の損益分岐点図表**

- グラフ概要: 横軸は参加人数(人)、縦軸は費用(千円)。費用線は50千円の席料を起点に、参加者1人につき飲食代3千円ずつ増加する。会費収入線は0人0円を起点に、参加者1人につき5千円ずつ増加する。
- 読み取り: 10人では会費収入50,000円、費用80,000円で30,000円の赤字。20人では会費収入100,000円、費用110,000円で10,000円の赤字。25人では会費収入と費用が125,000円で一致する。30人では会費収入150,000円、費用140,000円で10,000円の利益となる。

つまり、この忘年会の損益分岐点は25人であり、赤字を出さないためには参加者を25人以上集めればよいわけです。ここで描いた図表は、損益分岐点図表と呼ばれています。

### 2 費用は固定費と変動費に分けられる

#### 1 固定費、変動費とは

先ほどのケースでは、はじめに参加人数によらず一定の金額である席料と、参加人数によって金額が増減する飲食代という、性質の違う2種類の費用に分けて考えました。

企業活動で損益分岐点を算出する際も、これと同様に、売上高や操業度に関係なく一定に発生する費用と、売上高や操業度によって増減する費用に分けます。この2つの費用は、それぞれ固定費、変動費と呼ばれます。

|区分|意味|忘年会ケースの例|
|---|---|---|
|固定費|売上や操業度に関係なく一定に発生する費用|席料|
|変動費|売上や操業度によって増減する費用|飲食代|

#### 2 固定費と変動費の区分の方法

固定費と変動費について、販売業と製造業に分けて、もう少し詳しく確認しましょう。

##### 販売業の場合

販売業の場合、損益分岐点を把握するべき利益は、本業で得られる利益、つまり営業利益です。営業利益を算出するのに必要な費用について、損益計算書で確認し、各費用を固定費、変動費に分けます。

販売業の固定費と変動費は、つぎのように分けられます。

|区分|主な費用|
|---|---|
|固定費|減価償却費、人件費の一部、管理費の一部、営業費用の一部|
|変動費|売上原価、販促費、人件費の一部、管理費の一部、営業費用の一部|

減価償却費は売上高の増減があっても変化しない費用ですから、固定費です。また、売上原価、販促費は売上によって増減するため変動費です。

人件費は基本的には固定費ですが、残業代や繁忙期に増加するパート・アルバイトの費用は変動費になります。管理費やその他営業費用も同様で、基本的には固定費ですが、水道光熱費や通信費など売上の増減によって変動する費用も含まれています。このように明確に区分できない費用の場合、その実態に合わせて固定費と変動費へ配分する割合を決めることになります。

##### 製造業の場合

製造業では、売上高だけでなく、生産数量や稼働時間などの活動をもとに損益分岐点分析を行うことがあります。この場合、たとえば生産数量で分析を行うとすると、材料費、機械を動かす動力費、運搬費などが変動費となります。

これに対し、人件費は、同じ人員で生産し、残業代も増えないとすれば固定費です。また、光熱費、通信費などは、比率を使って固定費と変動費に分けます。このような費用は、準変動費と呼ばれます。

|区分|製造業での主な費用|
|---|---|
|固定費|減価償却費、事業税、租税公課、賃貸料、倉敷料、保険料など|
|変動費|外注費、動力費、運搬費、消耗工具費など|
|準変動費|事務消耗品、光熱費、修繕料、出張旅費、通信費、交際費、調査委託費など|

### 3 損益分岐点とその求め方

今度は先ほどのE販売店を例にとり、実際に損益分岐点の算出をしてみましょう。

#### 1 損益分岐点の図表から求める

はじめに、損益分岐点図表を描きます。損益分岐点図表は、つぎの順序で描きます。

|手順|内容|
|---|---|
|1|現時点の売上高の線を引く(線①)|
|2|固定費の線を引く(②固定費線)|
|3|費用の総額の線を引く(③総費用線)|
|4|売上高の線を引く(④売上高線)|

まず、E販売店の売上高は150,000円ですから、売上高150,000円の点で縦線①を引きます。つぎに、費用25,000円のところで、横線の②固定費線を引きます。売上高が0円のときの費用の総額は、固定費のみの金額である25,000円、売上高が150,000円のときの費用の総額は137,500円です。この2つの点を結んだ線が、費用の総額の線になります(③総費用線)。最後に、売上高0円と150,000円の点を結びます(④売上高線)。これで、損益分岐点図表が完成しました。

**図表3-5 | E販売店の損益分岐点図表**

- グラフ概要: 横軸は売上高(円)、縦軸は費用(円)。固定費線は25,000円で水平。総費用線は売上高0円・費用25,000円から、売上高150,000円・費用137,500円へ伸びる。売上高線は原点から売上高150,000円・費用150,000円へ伸びる。
- 読み取り: 総費用線と売上高線は売上高100,000円の位置で交わる。この点が損益分岐点であり、E販売店では売上高100,000円を境に黒字と赤字が分かれる。

#### 2 数式による損益分岐点の求め方

損益分岐点の算出は、損益分岐点の図表を描く方法のほかに、つぎのような公式にあてはめて計算することもできます。

$$損益分岐点売上高 = \frac{固定費}{1 - \frac{変動費}{売上高}}$$

先ほどのE販売店の数値をこれに当てはめて計算すると、つぎのようになります。

$$損益分岐点売上高 = \frac{25,000円}{1 - \frac{112,500円}{150,000円}} = 100,000円$$

ここまで、売上高と費用の関係についての損益分岐点をみてきました。損益分岐点は、売上高のほかに、販売数量で求めることもできます。損益分岐点販売数量は、つぎの計算式で求められます。

$$損益分岐点販売数量 = \frac{固定費}{販売単価 - 1個あたり変動費}$$

#### 3 限界利益による損益分岐点の計算

売上高と利益の関係について分析する方法として、固定費と変動費に分けて損益分岐点を求める方法のほかに、限界利益を使う考え方もあります。

限界利益とは、売上高から変動費を引いた金額です。また、売上高に対する限界利益の比率を限界利益率と呼びます。

$$限界利益 = 売上高 - 変動費$$

$$限界利益率(\%) = \frac{限界利益}{売上高} \times 100$$

固定費は売上高が0円であっても発生する費用ですから、利益をあげるためには、最低限、限界利益が固定費の金額を上回らなければなりません。

限界利益を使って損益分岐点売上高、損益分岐点販売数量を求める計算式は、つぎのようになります。

$$損益分岐点売上高 = \frac{固定費}{限界利益率}$$

$$損益分岐点販売数量 = \frac{固定費}{1個あたり限界利益}$$

ケース6の忘年会について、参加者1人あたりの限界利益はつぎのようになります。

$$参加者1人あたりの限界利益 = 5,000円(会費) - 3,000円(飲食代) = 2,000円$$

固定費は席料の50,000円です。赤字を出さない参加人数を求めるので、損益分岐点販売数量の計算式にあてはめると、つぎのようになります。

$$損益分岐点参加人数 = \frac{50,000円(固定費)}{2,000円(1人あたり限界利益)} = 25人$$

限界利益については、図表3-6のような限界利益図表を作成するとわかりやすいでしょう。

**図表3-6 | ケース6の限界利益図表** (※グラフ概要：横軸は参加人数(人)、縦軸は金額(千円)。固定費は50千円で水平線、限界利益は参加人数に応じて増える右上がりの線として表される。限界利益線が固定費線と交わる25人・50千円の点が損益分岐点。25人を超えると限界利益が固定費を上回る。)

損益分岐点と限界利益の考え方は、基本的には同じです。損益分岐点図表や計算式を使う企業もあれば、限界利益を使用する企業もあります。自社がどちらを使用しているか、確認してみましょう。

### 4 損益分岐点比率と安全余裕率

E販売店の例では、現在の売上高は150,000円であり、損益分岐点売上高は100,000円でした。これは現状の売上高に対して66.6%の比率であり、これを損益分岐点比率といいます。

損益分岐点比率の計算式はつぎのとおりです。

$$損益分岐点比率(\%) = \frac{損益分岐点売上高}{現状の売上高} \times 100$$

損益分岐点比率は、経営の安全性を表しています。E販売店の損益分岐点比率は66.6%ですから、売上高が現状に対して66.6%の水準に落ち込んでも赤字にはならないことを意味しています。

もう1つ、経営の安全性を意味する用語として安全余裕率が用いられることがあります。

安全余裕率の計算式はつぎのとおりです。

$$安全余裕率(\%) = 100 - 損益分岐点比率(\%)$$

つまり、損益分岐点比率の補数です。E販売店の安全余裕率は33.3%となり、これは現状よりも売上高が33.3%減少しても赤字にならないという余裕度を示しています。

では、損益分岐点比率は何%であれば経営が健全だといえるのでしょうか。もちろん損益分岐点比率は低ければ低いほどよいのですが、一般的にはつぎの数値が目安であると覚えておくとよいでしょう。

**図表3-7 | 損益分岐点比率の健全性の目安**

|損益分岐点比率|経営状態の目安|
|---|---|
|70%未満|高い利益率を確保できる、高収益経営の状態|
|70〜80%|よほど大きな環境変化がなければ赤字になることは考えにくく、優良経営の状態|
|90%以上|ライバル企業の動向や市場の変化によって赤字に転落するリスクがかなり高い、危険な経営状態|
|100%以上|赤字を計上している経営状態|

E販売店の損益分岐点比率は66.6%ですから、大変優良な状態ということができます。

### 5 原価に関する用語の整理

原価には、製造原価や売上原価、製造直接費や製造間接費、変動費や固定費などさまざまな用語があり、混乱する人も多いでしょう。ここでその関係を整理しておきましょう。

#### ①基本は3要素による製造原価と売上原価

工場の製造工程で発生するものが製造原価です。製造原価は、第2章で学習したように、原価の3要素と呼ばれる材料費、労務費、経費の3つから構成されます。

図表3-8をもとに、製造原価と売上原価の関係を確認しましょう。まず、製造原価は、完成品については倉庫に搬入され、在庫となります。期末(月末)にまだ完成していないものは、製造工程に残ったまま仕掛品の製造原価となります。

倉庫では、前期から繰り越された在庫 (①期首在庫)と当期の完成分(②当期完成分)が保管され、ここから払出を行っていきます。

期末になると、棚卸作業を行って倉庫にいくらの在庫が保管されているか(③期末在庫)を調査します。

**図表3-8 | 製造原価と売上原価**

●製造業の場合 製造工程 → (完成) → 在庫倉庫 → (販売分) → 売上原価

- 製造原価：材料費、労務費、経費
    
- 未完成分は仕掛品
    
- 在庫倉庫内：①期首在庫 ＋ ②当期完成分(製造原価) － ③期末在庫 ＝ ④払出分(売上原価)

## 3-2 目標利益を達成するには？

**学習のポイント**

- 損益分岐点分析は、採算計画や利益計画の立案に活用される。
    
- 利益をあげるための方策には、①売上高をあげる、②変動費を削減する、③固定費を削減するという方法がある。このうち固定費の削減では、変動費化することも有効である。
    

### 1. あといくら売上が増えれば黒字になる？

損益分岐点の分析は、さまざまな採算計算や利益計画に利用されます。この節では、それぞれの活用方法について、具体的にみていきましょう。

まず、損益分岐点分析の基本的な利用法の1つとして、いくら売上高を増やせば黒字になるのかを調べることがあげられます。

例として、現在の売上高が100百万円で営業利益が5百万円赤字であるF部門について考えてみましょう。

### 2. 目標利益を達成するための対策を検討する

目標利益を達成するには、損益分岐点売上高の計算式を応用して、必要な売上高や費用削減額を検討します。ここではG部門を例に、目標利益10,000千円を達成するための利益計画を立てます。

**①売上高の増加を検討する**

必要売上高は、固定費と目標利益を足した金額を限界利益率で割って求めます。

```text
必要売上高 = (固定費 + 目標利益) / (1 - 変動費 / 売上高)
```

G部門の数値をあてはめると、つぎのようになります。

```text
G部門の必要売上高 = (26,000千円 + 10,000千円) / (1 - 120,000千円 / 150,000千円)
                 = 180,000千円
```

目標利益10,000千円を達成するためには、180,000千円の売上高が必要です。売上高180,000千円は前年比20%増であり、現状では前年比2〜3%の増加をやっと確保している状態なので、非現実的な数値です。そこで、売上高目標は前年比8%増の162,000千円に設定します。

売上高が8%増加すると、変動費も8%増加します。この場合、営業利益は6,400千円となり、目標利益にはまだ3,600千円不足します。

**②変動費を削減する**

続いて変動費の削減を考えます。G部門では、利益率の高い商品を重点的に販売する、材料費の購入価格を引き下げる、販促費を削減する、といった対策によって変動費率を2%削減することにしました。

```text
変動費の削減額 = 売上高が8%増加した場合の変動費 129,600千円 x 削減率2%
              = 2,592千円
```

この結果、営業利益は6,400千円から2,592千円増加し、8,992千円となります。しかし、目標利益にはまだ約1,000千円不足しています。

**③固定費を削減する**

最後は固定費の削減です。固定費のうちの設備費は減価償却費や賃借料であり、部門ではコントロールできないものがほとんどです。このため、人件費や管理費、その他経費を削減するしかありません。G部門では社員のシフト体制を見直し、固定パート社員の人件費を一部削減することと、管理費やその他経費の削減で、1,008千円の固定費を削減することにしました。

**図表3-14 | 目標利益達成のための利益計画** (単位:千円)

|項目|前年実績|①売上高8%増加|②変動費2%削減|③固定費削減|
|---|---:|---:|---:|---:|
|売上高|150,000|162,000|162,000|162,000|
|変動費|120,000|129,600|127,008|127,008|
|固定費|26,000|26,000|26,000|24,992|
|費用合計|146,000|155,600|153,008|152,000|
|営業利益|4,000|6,400|8,992|10,000|

このように、損益分岐点分析の方法を応用することで、利益獲得のための計画立案に活用することができます。

### 3. 変動費と固定費を削減するには？

利益をあげるためには、売上高をあげるだけでなく、変動費や固定費を削減することが重要であることがわかりました。では、変動費や固定費を削減するにはどのような方法があるのでしょうか。

**①変動費の削減方法**

小売業などの商品を購入して販売する事業では、変動費は商品の購入代金や販促費用が中心です。このような事業の場合には、購入先と価格交渉を行う、価格の安い他の購入先からの購入を増やす、あるいは販促費用の活用方法を見直すなどの対策が必要です。

さらに進んで、購入先に対して製造工程や原価削減指導をして購入価格の引き下げを行っている企業、あるいはインターネットを活用したネット購買を導入している企業もあります。

製造業では、使用する材料費の仕入価格の削減や使用量の削減、また電力料や燃料費などの動力費の削減が必要です。

材料費の削減のためには、材料の歩留率や不良品の発生率の改善や、製品を構成している部品について機能とコストの関係を見直す価値分析(VA=バリューアナリシス)の手法なども効果的です。

動力費を削減するには、省エネの工夫や作業工程の見直し、製法の改善等を行って、製造時間や機械稼働時間の短縮を検討しなければなりません。

**②固定費の削減方法**

固定費には建物や機械の減価償却費や賃借料、保険料等の設備に関する費用があります。これらはいったん取得したり契約したりすれば、後から削減することが難しい費用であり、契約時の交渉がポイントになります。

人件費は、作業効率を高めて1人あたりの生産性を向上させ、相対的に抑制したり、パート社員化したりするといった削減方法が考えられます。

その他、外注化や業務委託等によって固定費を変動費化するという方法もあります。

**①固定費を変動費化するメリットとは？**

**②固定費を変動費化するためには**

さらに、自社で行っている業務を外部に委託して外注化すれば、仕事量に応じた支払いとなるため変動費となります。

- 作業員をパート社員にする
    
- 社用車をレンタカーとする
    
- 自社で行っている業務を外注化する

## 3-3 さまざまな採算計算にチャレンジしよう

**学習のポイント**

- 採算計算を的確に行うためには、その判断によって動くコスト(増分原価)と動かないコストの見極めが重要である。
    
- 外注化、原価以下の特別注文への対応、赤字製品の廃止などを検討する際には、利益の向上をめざすために、採算計算を行って意思決定をする必要がある。
    

### 1. 採算計算における原価とは？

企業活動では、「業務の外注化をするべきか」「特別注文に応じるべきか」「赤字製品を廃止するべきか」といった、採算性について検討しなければならない問題が数多く発生します。採算性については、売上、費用、利益の3つの視点から検討する必要があります。このとき有効なのが、変動費、固定費、限界利益を使って考える方法です。ここではまず、採算計算を行うための考え方について確認しておきましょう。

採算計算を行う際に使用する「原価」は、第2章で説明した製品の原価とは異なります。第2章で計算した製品原価は、材料費、労務費、経費のすべてを対象としていました。これを全部原価と呼びます。

これに対し採算計算では、なんらかの経営判断を行った場合に増加(または減少)する分の収入や費用だけを計算対象とします。その増加または減少分の原価を増分原価、減分原価と呼びます。また、全部原価に対して一部だけを計算対象とするため、部分原価と呼ぶこともあります。

さらに、製造原価の計算は、過去に発生した原価を計算対象にしますが、採算計算では、その判断や活動を行うことによって将来いくらのコストが増えるかを計算します。このため、対象となる原価は将来原価、あるいは未来原価と呼ばれることもあります。

そのほか、採算計算では、すでに支出が行われており、採算の判断に影響を及ぼさない原価を埋没原価と呼びます。

以上のような原価の種類を踏まえ、採算計算を学習していきましょう。まず、採算計算について感覚をつかむために、つぎの身近なケースについて、考えてみてください。

**【ケース7】** あなたは昨日、30,000千円の住宅Aを購入するため、手付金として5,000千円を支払いました。しかし、今日になって立地や住宅の規模、品質などがまったく同じであると評価できる住宅Bを見つけました。住宅Bの価格は24,000千円です。もし住宅Bを購入すると、住宅Aのために支払った手付金の5,000千円は返還されません。住宅A、住宅Bのどちらを購入したらよいでしょうか。

この判断をするにあたって、手付金の5,000千円が気になる人がいるかもしれません。

このケースの場合、住宅Aの手付金はすでに支払いが済んでいます。住宅Aを購入するにしても住宅Bを購入するにしても、この金額は返ってきません。したがって、手付金の5,000千円は、住宅A、住宅Bどちらを購入するのかという判断には関係がなく埋没原価となります。

このケースの判断を行う場合、「将来いくら支払うのか」という点だけを比較するべきなのです。つまり、意思決定することによって増える増分原価、将来払うことになる将来原価、追加で払う部分原価だけを考えます。この考え方をもとにすると、ここで検討すべき原価は、住宅Aの場合が25,000千円、住宅Bの場合が24,000千円です。このことから、住宅Bのほうが1,000千円得だ、ということができます。

ちなみに、それぞれの全部原価を計算すると、住宅Aが30,000千円、住宅Bはすでに支払った住宅Aの手付金5,000千円を加えて29,000千円となります。

### 2. 減価償却費の影響を理解する

**①減価償却費とは？**

企業が建物や機械、あるいは車両などの固定資産を購入した場合、その購入にかかった金額(取得価額)は、すべて一度に費用となるわけではありません。固定資産の使用期間の間、分割されて費用となります。この費用を減価償却費、使用期間を耐用年数といいます。

もし固定資産の取得価額を取得した年にすべて費用として計上すると、その年の利益が極端に少なくなってしまいます。また、それ以降の年度では利益が極端に大きくなり、毎年の利益が大きく変動してしまうでしょう。減価償却はこのような損益計算の変動を避けるために義務付けられています。

減価償却の方法には、定額法と定率法という2つの方法があります。定額法は毎年一定額を減価償却する方法です。これに対し定率法は、毎年一定率を減価償却します。どちらの方法を採用するかによって毎年の減価償却費は大きく異なってきますので、採算計算を行う場合にはその影響を正しく理解しておく必要があります。

なお、定額法と定率法のどちらを採用するかは企業の任意によりますが、いったん採用した方法は毎期継続して適用しなければなりません。

**②定額法と定率法による減価償却費の相違**

では定額法と定率法による毎年の減価償却費がどのように異なるか、つぎのケースでみてみましょう。

**【ケース8】** K社では、倉庫業務の効率化のため、運搬用の車両を2,000千円で購入しようと計画しています。この車両の耐用年数は5年であり、その期間にわたって年間500千円のコスト削減が見込まれています。また、車両を購入することによって毎年30千円の維持費が発生します。この運搬用の車両を購入すべきでしょうか。

**①定額法による場合**

定額法による場合、毎年の償却率はつぎのとおりです。

$$定額法による償却率 = \frac{1}{耐用年数(5年)} = 0.200$$

また、この償却率による毎年の減価償却費は、つぎのとおりです。

$$毎年の減価償却費 = 2,000千円(取得価額) \times 0.200(償却率) = 400千円$$

したがって、この車両を購入した場合、5年間にわたって毎年400千円を減価償却することとなります。この結果、利益は毎年70千円増加し、5年間では350千円の利益増となります。

|項目|1年目|2年目|3年目|4年目|5年目|合計|
|---|---|---|---|---|---|---|
|コストダウン額(利益)|500|500|500|500|500|2500|
|費用(減価償却費)|400|400|400|400|400|2000|
|費用(維持費)|30|30|30|30|30|150|
|費用計|430|430|430|430|430|2150|
|差引:利益増加額|70|70|70|70|70|350|

※減価償却では1円の備忘価額を残す必要があり、5年めの減価償却費は正確には399,999円となります。

**②定率法による場合**

定率法では、毎年一定率を減価償却します。その結果を示したものが図表3-18です。償却率の計算は少し複雑になるためここでは割愛いたしますが、このケースでは毎年0.400の定率(定額法の償却率の200%)で減価償却を行っています。

定率法では、はじめは減価償却費が大きく、徐々に減少していくことがわかります。このため、最初の年は定額法に比べて利益が出にくくなります。

このケースでも、5年間の合計はどちらの方法でも同じですが、定率法による場合には1、2年めは赤字となり、3年めになってやっと車両導入の効果が出ることになります。

### 3. 外注化は損か得か

現在は自社で製造しているものを、外注すると大幅にコストダウンできそうにみえる、ということはよくあります。しかし、実際に外注化を実施したところ、かえって費用が増大してしまったということもよくあるのです。

今回は、外注化を実施する場合の意思決定のポイントを確認しておきましょう。つぎのケースについて考えてみてください。

**【ケース9】** I製造部門では、社内のJ製造部門で生産した部品Yを加工して製品に仕立てています。これを他社に発注すると約20%も安く調達できることがわかりました。J部門における部品Yの原価などのデータはつぎのとおりです。なお、部品YはJ部門以外は使用していません。他社に発注したほうがよいでしょうか。

**図表3-19 | 部品の製造原価明細と外部購入価格** (単位:千円)

|項目|金額|
|---|---|
|直接材料費|600|
|労務費|180|
|経費(動力費)|130|
|経費(減価償却費)|100|
|経費(その他固定経費)|110|
|間接部門費配賦額|120|
|合計|1,240|

- 生産数量: 1,000個
    
- 1個あたり製造原価: 1,240円
    
- 外部購入価格: 1,000円
    

**①部品Yの製造原価の内訳**

図表3-19の製造原価明細を確認すると、J部門の部品Yに関する製造原価は1,240千円であり、1個あたりにすると1,240円です。これに対し、外部から購入する場合の1個あたりの価格は1,000円であり、J部門での製造原価と比較して約20%も安くなっています。

この原価明細を見る限り、外注したほうが得のように思えますが、もう少し詳しく考えてみましょう。

**②外注によって増える費用と減る費用は何か？**

部品Yの製造を外注した場合、1,000個分の購入額である1,000千円は確実に増加する費用です。これに対して減少する費用はいくらでしょうか。

製造原価のなかの材料費と動力費は変動費です。部品Yの生産がなくなれば、この合計である730千円は減少します。 しかし、固定費である労務費や減価償却費、その他固定経費は固定費であり、部品Yを製造しなくなってもなくなりません。また、間接部門費配賦額とは、工場の管理部門や購買部門などの間接部門の経費が割り振られたものです。このため、部品Yを製造しなくなっても工場全体として減少はしません。もし、部品Yを製造しなくなったとすると、他の製品がそれを負担することになるのです。

したがって、外注化によって増える原価と減る原価は図表3-20のようになります。

**図表3-20 | 外注化による増分原価と減分原価** (単位:千円)

|項目|増分原価|減分原価|
|---|---|---|
|外注費|1,000||
|直接材料費||600|
|動力費||130|
|合計|1,000|730|

増える原価は1,000千円、減る原価は730千円ですから、結果として270千円、原価が増加してしまうことがわかります。

このように、外注化をすれば費用が削減できるように思えても、短期的には費用が増加してしまうケースはよくあります。

外注化を実施するかどうかは、たとえば、部品Yの生産にかかわっている人や設備が他に転用できないかなど、長期的な視点で検討しなければなりません。

### 4. 原価以下の特別注文に応じるか

今度は、製造原価以下の価格での特別注文があった場合のケースを考えてみましょう。

**【ケース10】** K社では、つぎのような製品Zの製造を行い、特定のルートを対象とした販売を行っています。ある企業からつぎに示すような条件で特別注文があり、注文に応じるべきかどうか検討しています。注文に応じた場合、いくらの利益(または損失)がでるでしょうか。

■製品Zの現在の状況

- 生産数量(販売数量): 10,000個
    
- 販売単価: 1,000円
    
- 売上高: 10,000千円
    
- 製造原価: 7,000千円
    
    - ※うち変動費は5,000千円
        
    - ※1個あたりの製造原価は700円
        
- 物流販売コスト: 1,000千円
    
    - ※うち変動費は500千円
        
- 製造能力: 十分な余力あり
    

■特別注文の条件

- 数量: 3,000個
    
- 価格: 1個あたり600円
    
- 販売経費: 工場渡しであり、K社の費用は発生しない
    
    - ※なお、製品Zの用途は限定され、K社の既存の販売ルートに対する影響はない。
        

**●製造原価を分解すると**

このケースのような、製造原価以下の価格による受注について、多くの人は「利益がでるはずがない！」と感じるでしょう。1個あたりの製造原価が700円であるため、1個追加して製造するごとに700円余分に費用がかかってしまうと考えているからです。しかし、「原価のしくみ」を知っていれば、このケースのような特別注文に応じた場合に製造原価はどのように変化するか、正しい計算を行って的確な判断ができるのです。

では、「原価のしくみ」にもとづいて、このケースの解を求めていきましょう。まず、この特別注文に応じた場合、どんな費用がいくら増えるのか明確にします。

製品Zの製造原価は7,000千円ですが、このうち変動費は5,000千円、固定費は2,000千円です。物流販売コストも変動費と固定費に分解されますが、設問に「工場渡しで･･･」とあり、K社の物流販売コストは発生しないため、ここでは無視してよい費用となります。したがって、製造原価7,000千円だけに着目し、その内容を図表3-21に整理しましょう。

**図表3-21 | 製品Zの生産数量と原価の構造** (※グラフ概要：横軸に生産数量(個)、縦軸に費用(千円)。固定費2,000千円の上に変動費が乗り、10,000個の時点で総費用7,000千円。13,000個に増産した際の増分原価は1,500千円であることが示されている。)

**●特別注文による増分原価と増分収入、増分利益**

現在の生産数量は10,000個であり、固定費は2,000千円、変動費は5,000千円、1個あたりの変動費は500円です。製品Zの製造能力には十分な余力があるため、生産数量を増やしても固定費は増加しません。よって、このケースの特別注文に応じた場合に増加するのは、変動費だけです。

つまり、「特別注文に応じる」という意思決定をすることによって増える増分原価は3,000個×500円で1,500千円ということになります。一方、それによってもたらされる増分収入は1,800千円ですから、300千円、利益が増加(増分利益といいます)することになります。

このケースのポイントは、固定費の2,000千円は3,000個の特注に応じた場合にも増加しないため、この意思決定では関連しない固定費として扱うということです。厳密には、埋没原価はすでに支出済みで回収できない原価を指しますが、採算判断では、このように意思決定によって変わらない固定費を埋没原価と同じように「判断に含めない原価」として扱うことがあります。

**●受注決定の前提条件**

このケースのような特別注文では、受注価格が1個あたりの増分原価500円より高ければ利益が増加することになります。しかし、類似のケースで常に利益が出るとは限りません。「受注する」という意思決定をするためには、つぎの3つの前提条件が必要です。

**①製造能力に余力があること** 1つは設備や人員などの製造能力に余力があることです。もし製造能力に余力がなければ、特別注文に応じた場合、もう1つ製造ラインを作るための投資や、人件費などの固定費が余計にかかってしまうことになります。

**②既存のマーケットに影響を与えないこと** 特別注文に応じて販売した企業が、自社の販売ルートに対して自社の設定価格より安く販売することになったら、自社の売上高や利益は大幅に低下してしまいます。既存の販売ルートと特別注文のマーケットが互いに影響しないことが前提です。

海外に行くと、日本の製品が国内の販売価格より安く売られているのを見かけることがあります。これは、日本の市場では安定的に利益を確保できる価格で販売する一方で、製造能力の余力分を使って製品を製造し、変動費分だけを上回る価格設定で販売しているのです。ただし、その製品が海外から日本に逆流しないような価格設定が必要になります。

**③利益率よりも利益額が経営の優先課題であること** 先ほどのケースでは利益が300千円増加しましたが、利益率はつぎの計算のように、20%から約19.5%に低下しました。このような利益率の低下を経営的に容認できない場合は、利益率の低下に結びつく価格設定での特別注文は避けなければなりません。

|費用(製造原価)|7,000|8,500|

### 5. 赤字製品を廃止すると利益は増加するか

### 6. 全部原価計算と直接原価計算

この節で学習したケース9〜11では、製造原価について固定費と変動費に分けて考えることがポイントでした。第2章で学習した、材料費、労務費、経費のすべてを対象として計算した製造原価とは異なる原価の計算方法です。

第2章で学習した原価計算の方法は、全部原価計算と呼びます。外部へ報告する財務諸表の作成は、必ず全部原価計算で行わなければなりません。このため、全部原価計算は制度原価計算とも呼ばれます。

**●全部原価計算による損益計算の問題点**

この節で学習したように、全部原価計算には、生産数量や在庫数量が変動するような場合に利益の予測が難しくなってしまうという問題があります。この点について、つぎのZ社の事例をもとに、もう少し詳しく確認してみましょう。

**【ケース12】** Z社は、製品Aの製造・販売を行っています。製品Aの費用と販売額、3期間の生産数量、販売数量、期末の在庫数量はつぎのとおりでした。それぞれの期の損益はどのようになるのでしょうか。

(単位:個)

|項目|1期|2期|3期|
|---|---|---|---|
|生産数量|40,000|60,000|20,000|
|販売数量|40,000|40,000|40,000|
|在庫数量|0|20,000|0|

- 販売価格: 10円/個
    
- 製造変動費: 4円/個
    
- 製造固定費: 160,000円
    

Z社では、1期には製造した製品をすべて売り切っています。2期は生産数量60,000個に対して販売数量は40,000個であり、在庫が20,000個残ってしまいました。3期は残っていた在庫すべてと、それに加えてこの期で生産した20,000個も売り切ることができました。

このデータをみると、在庫を残した2期の業績が悪く、逆に残っていた在庫だけでなく生産した分も売り切った3期の業績がよいように思えます。しかし、結果は図表3-26のとおり、2期の利益が最大になりました。

なぜこのような結果になったのでしょうか。

2期は変動費と固定費の合計である400,000円の製造原価を投入し、60,000個の製品を完成しました。1個あたりの製造原価は約6.7円です。

しかし、損益計算に使われるのは売上原価です。売上原価は、その期に売上になった分の製造原価のことでした。2期は、60,000個製造したうち40,000個払出したのですから、売上原価は400,000円×40,000個÷60,000個=266,667円となります。つまり、2期は生産数量を増やしたため1個あたりの製造原価が小さくなり、製造原価のうち在庫分が翌期の売上原価として差し引かれるため、売上原価が下がったのです。2期の1個あたりの製造原価が小さくなった原因は、大量に製造したため、1個あたりの固定費の負担が小さくなったからです。 このように、全部原価計算では、生産数量の増減があった場合に1個あたりの固定費負担が変化することや、在庫数量の増減があった場合に売上原価に影響を与えるため、損益の予測が大変難しくなります。

**●売上高の増減による損益の変化がわかる直接原価計算**

企業や部門の内部で損益の管理を行いたい場合は、全部原価計算ではなく、直接原価計算によって損益計算を行うことがあります。直接原価計算とは、変動費のみを製品の原価とし、固定費は期間の費用として計算する方法です。先ほどのZ社のケースでは、直接原価計算で損益計算を行うと、全部原価計算の結果とは異なり、営業利益は3期間とも同じになります(図表3-27)。

直接原価計算では、売上高から変動費を差引いて限界利益(または貢献利益)を求めます。限界利益は、変動費のみを原価として計算するため、売上高の増減に比例した金額となります。この利益は、固定費や販売管理費が一定であれば売上高の増減に応じて増減することになり、業績の予測がしやすいといえるでしょう。

このような直接原価計算は、外部報告の制度としては認められていませんが、内部管理の目的で多くの企業で採用されています。

# 第3章 まとめ

## 3-1 損益分岐点はいくらか？

- 売上と費用が一致する売上高を損益分岐点と呼び、獲得する利益の目標を立てるために利用される。損益分岐点売上高は、つぎの式で計算される。

```
損益分岐点売上高 = 固定費 / (1 - 変動費 / 売上高)
```

- 売上高から変動費を引いた金額を限界利益と呼び、限界利益を使って損益分岐点を求める方法もある。つぎの式で計算される。

```
損益分岐点売上高 = 固定費 / 限界利益率
```

- 経営の安全性を確認するために、損益分岐点比率が用いられる。損益分岐点比率はつぎの式で計算され、90%以上であると危険であるとされる。

```
損益分岐点比率 = 損益分岐点売上高 / 売上高 x 100
```

## 3-2 目標利益を達成するためには？

- 目標とする利益を達成するために必要な売上高は、つぎの式で計算される。

```
必要売上高 = (固定費 + 目標利益) / (1 - 変動費 / 売上高)
```

- 固定費を変動費化すると、安全余裕率が高まり、経営のリスクを減少させることができる。

## 3-3 さまざまな採算計算にチャレンジしよう

- 採算計算では、全部原価ではなく、増分原価、減分原価のみを計算対象とする。また、採算の判断に影響のない原価は埋没原価と呼ばれる。

- 利益の予測を行う際は、変動費のみを原価とする直接原価計算が用いられることがある。

# 第4章 業務の分析と改善にチャレンジ

計数分析の目的は、最終的にコスト削減や利益の向上を行うことです。そのためには、ビジネスの活動、すなわち業務の進め方を改善しなくてはなりません。

この章では、今後、業務の改善にチャレンジしていくために必要な業務の分析手法や、業務改善の進め方を学習します。

## 4-1 業務を「見える化」するためには

**学習のポイント**

- 業務改善は、現状の業務を「見える化」することからスタートする。
- 業務の現状分析には、ABC分析、ポートフォリオ分析、ワークサンプリング手法、アンケート手法などが利用される。

### 1 業務改善の進め方

#### 1 業務改善のプロセス

この章では、これまで学習してきた計数知識を活かして、業務改善を行う方法を学習します。業務改善は、①業務の実態把握、②問題点の把握、③改善案の立案、④改善案の実施・定着化という手順で進めます。

**図表4-1 | 業務改善のプロセス**

|段階|内容|
|---|---|
|業務の現状分析 ①業務の実態把握|どんな業務が、いつ、誰によって、どのように、どれだけ行われているかを把握する。|
|業務の現状分析 ②問題点の把握|問題は何か、なぜその問題が発生したか、その根本的な原因は何かを把握する。|
|③改善案の立案|問題に対して的確な改善策を考え、だれがどのように改善を進めるか、スケジュールを立てる。|
|④改善案の実施・定着化|改善策を実施し、フォローアップを行う。|

業務改善とは、仕事のプロセスを見直し、よりよい進め方に変えていくことです。したがって、はじめに現状の業務がどのように行われているか、その実態を把握しなければなりません。どのような業務が、いつ、だれによって、どのような方法・手順で行われているかを明確にするのです。この業務の実態把握は、業務を「見える化」する作業といい換えてもよいでしょう。

ここが、業務改善のプロセスのなかでもっとも重要であり、かつ難しいプロセスです。業務の実態把握が正確にできていなければ、つぎのステップである問題点の把握も的確にできません。また、①業務の実態把握と②問題点の把握を合わせて、現状分析と呼びます。

現状分析を的確に行うことができたら、つぎは③改善案の立案です。改善案の立案は、思いつきに頼ってもうまくいきません。ここで重要になるのは、ほかの改善事例についての情報収集や、発想力です。情報収集をしっかり行い、よいアイデアを出すことができたら、中心となって進める担当者、責任者を決定し、スケジュールを含めた改善案をまとめます。

最後は④改善案の実施・定着化です。改善案がどんなに綿密なものだったとしても、実際に始めてみると計画どおり進むことはほとんどありません。このステップでは、粘り強い改善案の手直しが求められ、最後までやり遂げる根気や信念が必要です。

#### 2 業務の実態把握「見える化」の重要性と難しさ

先ほども述べたとおり、業務改善にとって業務の実態把握はもっとも重要であるにもかかわらず、大変難しいプロセスです。それは、実際の業務には見て確認するだけでは何の作業を行っているかわからないものや、毎日異なる人数で作業が行われているものなど、どのような調査を行えば正確な把握ができるのか、わかりにくいものがあるからです。

業務の実態を「見える化」するには、業務の特性に合わせた現状分析の手法を使う必要があります。

ここからは、業務の実態把握のために使われる現状分析の手法として、①ABC分析、②ポートフォリオ分析、③ワークサンプリング手法、④アンケート調査という4つの分析手法について、それぞれの特徴と具体的な方法を説明します。

### 2 現状分析の手法

#### 1 ABC分析

ABC分析とは、構成比の高いものに着目し、優先的に対策を行うことで、より少ない時間で最大の効果を得ようという意図で使われる、重点管理の手法です。現状分析の手法としてもっとも基本的なものだといえます。

構成比を分析した結果、数量や頻度の多いものをAランク、やや多いものをBランク、少ないものをCランクと分類するため、ABC分析といわれます。小売業では、売上高がゼロの商品もあり、これをZランクと名付けてABCZ分析ということもあります。

営業所全体の売上高を維持しようとするなら、Aランクの3社をもっとも重視します。分析結果の数値どおりにみれば、この3社に営業所のパワーの65%を投入してもよいということになるでしょう。

この例では、ABC分析を得意先の重点管理のために活用していますが、ほかにも、たとえばクレーム対策の重点管理などにも活用できます。クレームの種類と発生件数を分析し、件数の多い順に並べ、60%までのものをAランクとしたとしましょう。Aランクのクレームが3種類だった場合、3種類をなくすことによって、全体の60%のクレームが解消することが見込まれます。

このようにABC分析は、現状を整理して重点管理を行うためのツールとして活用されます。

## 4-2 業務改善の実践

### 学習のポイント

- 業務改善を実施するには、さまざまな先進事例の情報収集が重要である。
- 在庫改善では、適正在庫、リードタイム、安全在庫を把握する必要がある。
- 製造工程の改善では、ボトルネックとラインバランス効率を分析する。
- 作業プロセスと分担の改善では、もっとも重要な作業により多くの時間をかけられるように検討する。

### 1 ベンチマーキング手法とは？

前節で、業務改善のステップのうち改善案の立案の段階では、よいアイデアを出すために情報収集と発想力が重要であることを説明しました。情報収集とその活用のしかたとしては、ベンチマーキング手法が有効です。

ベンチマーキング手法とは、業務改善などの計画立案にあたって、パフォーマンスのよい他部門や同業他社、異業種企業、世界のエクセレントカンパニーなどの事例を徹底的に研究し、ベストプラクティス（最善の方法）を把握して、それを参考にしようという方法です。意訳すれば「先進事例研究」といえるでしょう。業績のよい部門や企業には、ほかにはないノウハウが必ずあり、多くの場合、ほかとはまったく異なった発想で仕事が行われているものです。業務改善を行うにあたっては、自社、自部門のこれまでの経験から生まれる発想だけに頼るのではなく、ベンチマーキングを行うことが非常に重要であるといえるでしょう。

ベンチマーキング手法を行う場合、ベンチマーキングの対象は、自社内の部門はもちろん、同業他社や異業種企業、世界のエクセレントカンパニーまで含めて考えなければなりません。このような対象を見つけるためには、ベストプラクティスの情報に対して、常にアンテナを高くしておくとよいでしょう。

### 2 在庫改善

この節では、代表的な業務改善の例として、在庫、製造工程、作業プロセスとその分担についての改善を事例をもとに学習します。ここでは、まず在庫改善の例をみてみましょう。

**【ケース14】** つぎの特性をもつ商品Aについて、発注日の適正在庫は何個でしょうか。

**商品Aの特性**

- 発注から納品までのリードタイム: 2日
- 1日あたりの平均販売数量: 4個
- 安全在庫数量: 4個

在庫改善を実施するためには、まず適正在庫を決定する必要があります。適正在庫とは、つぎの2つの条件を満たした在庫数量のことです。

**適正在庫の条件**

- 品切れを起こさないこと
- 各品目が販売数に応じた数量であること

また、商品を発注してから納品されるまでには、通常は数日間のリードタイムがあります。発注を行う日にはリードタイムの間に販売されるであろう数量の在庫がなければ、納品されるまでに欠品が生じてしまいます。さらに、納品の直前に在庫が0になっていると、販売数量が予測より多かった場合、欠品になってしまう可能性があります。このため、万一の場合にそなえて、余分な在庫も持たなければなりません。これを安全在庫と呼びます。

以上の考え方で、発注日における適正在庫を計算すると、つぎのような式になります。

```text
発注日の適正在庫 = リードタイムの日数 x 1日あたりの販売数量 + 安全在庫
```

先ほどの式に商品Aの数値をあてはめると、つぎのような計算になります。

```text
商品Aの発注日の適正在庫 = 2日 x 4個 + 4個 = 12個
```

**図表4-10 | 商品Aの在庫数の推移**

グラフは、縦軸を在庫数(個)、横軸を日数(日)として、商品Aの在庫数が販売により日々減少し、納品日に補充される様子を示しています。安全在庫は4個で、発注日の適正在庫数は12個です。商品Aは1日あたり4個販売されるため、発注日から納品までの2日分8個と安全在庫4個を合わせた12個が、発注日に必要な在庫数となります。

このように、在庫改善を行う場合には、商品や材料等について一品一品適正在庫数を割り出す必要があります。なお、在庫管理の手法は、数量ではなく金額をもとに行うものもあります。みなさんの会社や部署ではどのような在庫管理を行い、どんな発注方法をとっているのか、一度確認しておくとよいでしょう。

### 3 製造工程の改善

今度は、製造工程の改善事例をみてみましょう。製造工程の効率化を行うための典型的な例として、ボトルネックとラインバランス効率の分析があります。ラインバランス効率とは、ライン(工程)の効率性をみる指標のことです。また、ボトルネックとはラインバランス効率を阻害する原因のことです。

ラインバランス効率はつぎの計算式で表されます。

```text
ラインバランス効率 = 作業時間の合計 / ボトルネックによる拘束時間
```

この指標は100%に近いほどよい状態であるといえます。では、完成までの作業にAからFまでの6つの工程がある製品Bについて、製造工程の分析を行ってみましょう。工程別の実際所要時間は、図表4-11のとおりです。

**図表4-11 | 製品Bの工程別実際所要時間**（単位: 時間）

|工程番号|A|B|C|D|E|F|合計|
|---|---:|---:|---:|---:|---:|---:|---:|
|時間|3|4|5|8|5|4|29|
|作業者数|1名|1名|1名|1名|1名|1名|6名|

**1 ラインのボトルネックを把握する**

図表4-11を見ると、6つの工程の作業者数はすべて1名で、D工程に一番時間がかかっていることがわかります。このことから、この作業においてはD工程が「ボトルネック工程」ということになります。

各工程の作業時間の差からすると、現状のままではB、C、D工程の前に仕掛品の山ができ、逆にE、Fの作業者は手待ち(アイドリング)の状態が発生します。かといって、作業時間の比較的短いA、B、Cの工程で作業者がほかの工程の様子をみながら仕掛品を多く出さないようにゆっくりと仕事をすれば、D工程以外のすべての工程で手待ちが発生してしまいます。まさにムダの発生です。

**2 ラインバランス効率を把握する**

ではラインバランス効率を測定してみましょう。図表4-11によれば、作業時間の合計は29時間です。一方、ボトルネックによる拘束時間は、6人全員がボトルネック工程のD工程に拘束されたため、8時間 x 6人 = 48時間となります。これを計算すると、ラインバランス効率はつぎのようになります。

```text
ラインバランス効率 = 29時間 / 48時間 = 約60.4%
```

図表4-12は、ラインバランスを図表で表したものです。色のついた部分が、実際に作業を行った時間です。一方、白い部分がアイドルタイムです。

**図表4-12 | 現在の製品Bのラインバランス図表**

グラフは、縦軸を時間、横軸を作業者として、A=3時間、B=4時間、C=5時間、D=8時間、E=5時間、F=4時間の棒で作業時間を示しています。D工程の8時間がボトルネックとなり、6人全員が8時間拘束されるため、合計48時間のうち実作業時間29時間、アイドルタイム19時間が発生しています。

**3 スピードを上げる**

ラインバランス効率を上げるためには、D工程の作業スピードを高めなければなりません。作業動作の分析を行い、ムリ、ムダ、ムラを徹底して取り除くことが必要でしょう。しかし、このような作業方法の改善に限界がある場合には、D工程をもう1人増員することで2工程に分解すると大幅なスピードアップが実現できます。

## 4-3 販売業務の分析と改善

### 学習のポイント

- 小売業等では、売上高は客数 x 客単価に分解でき、それぞれの傾向を把握したり、予算との対比を行う必要がある。
- 拡販のための施策立案には、ポートフォリオ分析が有効である。
- 売上に季節変動がある場合、移動合計値を活用した販売予測が有効である。
- 各月の販売計画を立案するとき、季節指数にもとづいて目標を設定する必要がある。

### 1 小売業における売上高の分析方法

スーパーマーケットやコンビニエンスストアなどの小売業では、不特定多数のお客さまに膨大な種類の商品を販売します。したがって、売上高についてその内容を分析したくても、お客さまごとに分析することは難しく、「何人のお客さまがいくら商品を購入したか」という視点で分析することが基本です。ここでは、ABCZ分析などの手法を活用して、小売店の販売業務についての業務分析と改善を考えていきましょう。

#### 1 客数、客単価の分析

売上高は「客数」x「客単価」で決まります。客数は実際に売上があった客数（売上客数）、客単価はお客さま1人あたりの平均購入額です。また、客単価は1人のお客さまが平均いくつの商品を購入したかを示す「買上点数」と平均いくらの商品を購入したかを示す「1点単価」に分解されます。客数は、来店したお客さまの総数である「来店客数」と、そのうち商品を購入したお客さまの比率である「買上率」に分解されます。

**図表4-16 | 小売業の売上分析の基本**

$$売上高 = 客数 \times 客単価$$

$$客数 = 来店客数 \times 買上率$$

$$客単価 = 買上点数 \times 1点単価$$

（※イラストコメント：客数と客単価に分解して分析するのね！）

では、図表4-17のような実績のQ店の売上高について、内容を分析してみましょう。

|買上点数|3.2|3.0|93.8%|
|1点単価|468.8|466.7|99.6%|

### 3. 販売予測と季節変動の分析

企業が経営計画を立案する場合、出発点になるのが販売予測です。販売予測とは売上高を予測することですが、経営者・管理者だけで考えるものではなく、購買、製造、営業、研究開発など、すべての担当者が認識しておく必要があります。

販売予測にもとづいて会社全体の売上目標が決まると、つぎにそれが各部門の目標に落とし込まれます。各部門では、担当する商品や地域ごとに、いつまでにどのくらいの数量を作るのか、あるいは売上をどのくらいあげればよいかなどの計画を立てます。つまり、販売予測はあらゆる部門の計画のスタートとなるものであり、できるだけ正確に行うことが重要です。

#### 1. 季節指数の算出とその活用方法

「2月と8月は商売が暇な時期」といわれることがありますが、どんな事業でも売上高は季節によって変動します。売上高を分析して好調に推移しているかどうかを判断する場合は、季節変動の影響を把握しておかなければなりません。

売上高の季節変動の影響を分析する基本的な方法として、季節指数を求めるものと、月度の移動合計値を求めるものがあります。

**①季節指数の重要性**

月々の売上目標を設定するとき、「前月対比で○○%の売上増加」という表現をすることがあります。しかし、売上に季節変動がある場合には、これでは正しい目標とはいえません。

季節変動を見込んだ正しい売上目標を立てるためには、過去の売上データから毎月の売上変動傾向を「季節指数」として算出し、月ごとの売上予測を行う必要があります。また、季節指数は仕入数量の調節や、従業員の配置・ローテーションにも役立てることができます。

このように、季節指数は売上の季節変動がある事業を行っている場合には重要な指標です。

**②季節指数の求め方**

季節指数を算出する方法には、統計学を活用するものもありますが、ここでは過去の実績にもとづいて算出する方法を学習します。過去の実績から算出する場合、まず過去何年間の実績を用いるかを決めます。精度を高めるためには最低3年間の実績は必要です。ここでは、4年間の実績を用いる例で学習します。

**図表4-26 | 4年間の販売実績による季節指数** (単位:個)

|月|20X1年度|20X2年度|20X3年度|20X4年度|月合計|季節指数|
|---|---:|---:|---:|---:|---:|---:|
|4月|123|125|132|145|525|7.2%|
|5月|146|146|155|175|622|8.5%|
|6月|120|135|145|158|558|7.7%|
|7月|142|145|60|176|523|7.2%|

|9月|150|152|160|179|641|8.8%|

図表4-26は、4年間の販売実績による各年の季節指数です。季節指数は、4年間の月ごとの販売数量合計である「月合計」を4年間の合計販売数量で割って求めます。たとえば、4月の季節指数はつぎのような計算になります。

```text
4月の季節指数 = 4月の月合計(525個) / 4年間の合計(7,284個) x 100 = 7.2%
```

同様に各月の季節指数を算出し、過去4年間における各月の売上変動傾向を明確にします。

**③季節指数を利用した販売目標の立てかた**

季節指数を算出できたら、今後はその値を将来の年間売上目標額に掛け合わせ、各月の販売目標値を立てます。

たとえば、20X5年度に前年比で10%の販売数量増加をめざす場合には、20X5年度の年間販売目標は20X4年度の年間販売実績2,085個に10%増を上乗せして、2,294個となります。

```text
20X5年度の年間販売目標 = 20X4年度の年間販売実績(2,085個) x 110% = 2,294個
```

このように年間販売目標に季節指数をかけたものが月別販売目標となります。4月の販売目標数量を計算すると、つぎのようになります。

```text
4月の販売目標数量 = 2,294個 x 4月の季節指数(7.2%) = 165個
```

（※イラストコメント: 季節指数を使えば予測がより正確になるんだね）

#### 2. 移動合計値による販売予測

製品の販売計画や生産計画を立案する場合、今後その製品がいくつ売れるのか、販売予測を行うことからスタートします。しかし、実際には年度の傾向や季節変動などがあり、販売予測は大変難しいものです。数量を読み違えて材料の購買発注や生産をすると、大量の過剰在庫を抱え込むことになります。また逆に、予測を控えめにしすぎると機会ロスが発生してしまいます。

販売予測の方法には、回帰分析などの統計手法もありますが、ここでは小売業で伝統的に用いられてきた手法であり、だれでも簡単にできる移動合計値による方法について説明します。

このZチャートは、つぎのようなパターンがあり、販売の傾向を見える化するために大変便利な手法です。

**Zチャートのパターン**

- **Zの形を見る**
    
    - ①Zが右上がり：売上は増加傾向
        
    - ②Zが並行：売上は横ばい
        
    - ③Zが右下がり：売上は減少傾向
        
- **Zの幅を見る**
    
    - ④Zの幅が広い：売上が好調
        
    - ⑤Zの幅が狭い：売上は低調
