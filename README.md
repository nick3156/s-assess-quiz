# s-assess-quiz

Sアセス一次試験対策の単一HTMLクイズアプリ。複数ソース（教科書・中期経営計画・文字起こしなど）から出題、カテゴリ別出題・苦手重点モード・ダークモード対応。

公開URL: https://nick3156.github.io/s-assess-quiz/

## 状態 (2026-04-24)

**一次試験は 2026-04 に合格済み、このアプリの用途は完了**。
2026-05 の二次試験は形式が異なる (筆記・論述系) ためこのクイズ形式では対応不可。保存用としてリポジトリは残す、積極更新は予定なし。

## 特徴

- 300問以上（7カテゴリ）を動的出題
- 全問網羅保証：出題済みを localStorage に永続化
- 苦手重点モード：間違えた問題を優先的に再出題
- カテゴリ選択の即時反映・リロード復元
- 5択問題＋選択肢数の柔軟化（3〜5択）
- ダークモード・連続正解ストリーク表示

## ファイル

- `index.html` — クイズ本体（単一HTML）

## ローカルで動かす

ブラウザで直接開くだけで動作。ビルド不要。

```
open index.html
```
